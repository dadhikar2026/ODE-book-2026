    Section 5.5 A Convolution Theorem and its Applications

    A crucial step in solving a linear initial value problem using Laplace transforms is determining the inverse Laplace transform of the solution’s Laplace transform. This step can become challenging when the expression, whose inverse Laplace transform is desired, is complicated. In this section, we discuss a procedure with which we can determine the inverse Laplace transform of an expression that is a product of known Laplace transforms. For example, we recall that
    \begin{equation*} \mathcal {L}^{-1}\left\{\ds\frac{1}{s^2}\right\} = t \quad \text{ and }\quad \mathcal {L}^{-1}\left\{\ds\frac{2}{s^3}\right\} = t^2. \end{equation*}
    However, if we were to compute
    \begin{equation*} \mathcal {L}^{-1}\left\{\ds\left(\frac{1}{s^2}\right) \left(\frac{2}{s^3}\right)\right\}, \end{equation*}
    a direct way is to see this problem as
    \begin{equation*} \mathcal {L}^{-1}\left\{\frac{2}{s^5}\right\} = \frac{2}{4!}\; t^4 = \frac{t^4}{12}. \end{equation*}
    This leads to a natural question: in what way can we combine \(t\) and \(t^2\) to produce \(t^4/12\text{?}\) The operation for combining inverse Laplace transforms—specifically, to determine the inverse of a product—is known as convolution, which is formalized in the following definition.
    🔗

    Definition 5.5.1.

    Let \(f\) and \(g\) be functions on \([0, \infty).\) The convolution of \(f\) and \(g\text{,}\) denoted by \(f\ast g\text{,}\) is the function defined by
    \begin{equation*} (f\ast g)(t) = \int_0^t f(u) g(t-u)\, du \end{equation*}
    whenever the integral exists for a given \(t\) in \([0, \infty).\)
    🔗
    🔗
    For computational purposes, \((f * g)(t)\) may also be written as \(f(t) * g(t)\text{.}\) It can be proved that \(f\ast g = g\ast f;\) see, for example, Item 5.5.3.2.ii in Exercises 5.5.3. The convolution \(1\ast f\) is particularly useful, and it is given by
    \begin{equation*} (1\ast f)(t) = \int_0^t f(u) \, du. \end{equation*}
    See, also, Item 5.5.3.2.v in Exercises 5.5.3.
    🔗

    Example 5.5.2.

    Let us return to the example discussed before Definition 5.5.1 and calculate the convolution of \(f(t) =t\) and \(g(t) =t^2\text{.}\) We have
    \begin{equation*} \begin{split} (f\ast g)(t)& = \int_0^t u (t-u)^2\, du\\ &= \int_0^t (t^2u-2tu^2 +u^3)\, du\\ &= \left[t^2\frac{u^2}{2}-2t\frac{u^3}{3}+\frac{u^4}{4}\right]_0^t\\ &=\frac{t^4}{2}-\frac{2t^4}{3}+\frac{t^4}{4}\\ &=\frac{t^4}{12}, \end{split} \end{equation*}
    as expected.
    🔗
    🔗

    Subsection 5.5.1 Convolution Theorem

    We now state and prove the most important theorem of this section.
    🔗

    Theorem 5.5.3.

    Let \(f\) and \(g\) be piecewise continuous functions on \([0,\infty)\) and of the exponential order \(\alpha\) as \(t\to\infty\text{.}\) Then
    \begin{equation*} \mathcal {L}\{(f\ast g)(t)\}= F(s) \,G(s); \end{equation*}
    or equivalently,
    \begin{equation*} \mathcal {L}^{-1}\left\{ F(s) G(s)\right\} =\mathcal {L}^{-1}\{F(s)\}\ast \mathcal {L}^{-1}\{G(s)\} = (f\ast g)(t) \end{equation*}
    for \(s>\alpha.\)
    🔗
    🔗
    To see how this theorem holds, we need to use a method for changing the order of an iterated integral. We include a proof of the Convolution Theorem below for the sake of completeness; however, not to be distracted from the proof, the reader may choose to skip it for now and proceed directly to apply the theorem.
    🔗

    Proof of Theorem 5.5.3.

    We recall that
    \begin{equation*} F(s) = \int_0^\infty e^{-s\eta} f(\eta) \, d\eta\quad \text{ and }\quad G(s) = \int_0^\infty e^{-su} g(u) \, du. \end{equation*}
    Then
    \begin{equation*} \begin{split} F(s) G(s) & = \int_0^\infty e^{-s\eta} f(\eta) \, d\eta\; \int_0^\infty e^{-su} g(u) \, du\\ &=\int_0^\infty g(u) \left( \int_0^\infty e^{-s(\eta+u)} f(\eta) \, d\eta \right) \, du\\ & = \int_0^\infty g(u) \left( \int_u^\infty e^{-st} f(t - u) \, d\xi \right) \, du, \end{split} \end{equation*}
    where \(t=\eta+u.\) By changing the order of integration, we obtain
    \begin{equation*} \begin{split} F(s) G(s)&=\int_0^\infty e^{-s t} \left( \int_0^t g(u) f(t - u) \, du \right) \, dt \\ &=\int_0^\infty e^{-st} (g\ast f)(t)\, dt\\ & = \mathcal {L}\{(f\ast g)(t)\}. \end{split} \end{equation*}
    🔗
    🔗
    Some fundamental properties of the convolution operation are discussed in Exercises 5.5.3; see the problem Item 5.5.3.2.ii.
    🔗
    🔗

    Subsection 5.5.2 Application of the Convolution Theorem to Initial Value Problems

    The following theorem concerns an interesting application of Theorem 5.5.3 to linear initial value problems that involve the Dirac delta functions.
    🔗

    Theorem 5.5.4.

    Let \(y_\delta(t)\) denote the solution of the initial value problem
    \begin{equation} ay''+by'+cy = \delta(t), \quad y(0) = 0, y'(0) = 0.\tag{5.5.1} \end{equation}
    The the solution \(y(t)\) of the initial value problem
    \begin{equation} ay''+by'+cy = g(t), \quad y(0) = 0, y'(0) = 0\tag{5.5.2} \end{equation}
    where \(a, b, c\) are constants with \(a\ne 0,\) and \(g\) is a piecewise continuous function that is of exponential order as \(t\to\infty,\) is given by
    \begin{equation*} y(t) = (y_\delta \ast g)(t). \end{equation*}
    🔗
    🔗
    We first remark that \(y'\) and \(y''\) in (5.5.1) are to be interpreted differently (as distributional derivatives) because of the presence of \(\delta(t)\) on the right side of the equation. The fact beyond the scope of this book is that the Laplace transforms of distributional derivatives coincide with the Laplace transforms of the classical derivatives.
    🔗

    Proof of Theorem 5.5.4.

    We first find the solution of the initial value problem (5.5.1). Applying the Laplace transform \(athcal{L}\) to the initial value problem (5.5.1) and using \(y(0)= y'(0)=0\) yields
    \begin{equation*} (as^2+bs +c)Y_\delta(s)= 1, \end{equation*}
    that is,
    \begin{equation*} Y_\delta(s)= \frac{1}{as^2+bs +c}. \end{equation*}
    Then, by taking the inverse Laplace transform, we have
    \begin{equation*} y_\delta(t) = \mathcal{L}^{-1}\left\{\frac{1}{as^2 + bs + c}\right\}. \end{equation*}
    Next, we take the Laplace transform of both sides of the equation in (5.5.2) to get
    \begin{equation*} Y(s) = \frac{G(s)}{as^2 + bs + c} = G(s) Y_\delta(s). \end{equation*}
    Finally, by applying the convolution theorem (Theorem 5.5.3), we conclude that
    \begin{equation*} y(t) = (y_\delta \ast g)(t) \end{equation*}
    solves the initial value problem (5.5.2) .
    🔗
    🔗

    Example 5.5.5.

    Consider the initial value problem \(y''+ y= e^t,\; y(0) = y'(0)= 0.\) To solve this problem, we first find the solution of the initial value problem \(y''+ y= \delta(t),\; y(0) = y'(0)= 0.\) Applying the Laplace transform to this problem yields
    \begin{equation*} (s^2 + 1) Y_\delta(s) = 1, \end{equation*}
    that is,
    \begin{equation*} Y_\delta(s) = \frac{1}{s^2 + 1}. \end{equation*}
    Then, by taking the inverse Laplace transform, we have
    \begin{equation*} y_\delta(t) = \sin t. \end{equation*}
    Finally, by applying Theorem 5.5.4 with \(g(t) = e^t\text{,}\) we conclude that
    \begin{equation*} y(t) = y_\delta(t) \ast e^t = \sin t \ast e^t = -\frac{1}{2}\left(-e^t+\sin t+\cos t\right) \end{equation*}
    on \((-\infty, \infty).\)
    🔗
    🔗
