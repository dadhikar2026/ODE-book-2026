    Section 5.4 Laplace Transforms of Periodic Functions

    In Section 5.1, we computed the Laplace transforms of the periodic functions \(\sin(\omega t)\) and \(\cos(\omega t)\) that are continuous. In applications, we also have to address periodic functions that are periodic but piecewise continuous on \([0, \infty).\)
    🔗

    Subsection 5.4.1 Periodic Functions

    🔗

    Definition 5.4.1.

    A function \(f\) is said to be periodic on \((-\infty, \infty)\) if there exists a number \(T>0,\) called a period of \(f,\) such that
    \begin{equation*} f(t) = f(t+T)\quad\text{ for all } t\in (-\infty, \infty). \end{equation*}
    🔗
    🔗
    Graphically, a periodic function \(f\) repeats the portion of its graph restricted to \([0, T]\) across subsequent intervals of the same length \(T\text{.}\)
    🔗
    A piecewise continuous periodic function \(f(t)\) with a period \(T\) has the following property:
    \begin{equation*} f(t) = f(t+T) = f(t+2T) = \cdots = f(t+nT) \end{equation*}
    for all positive integers \(n.\) This property is visualized in Figure 5.4.2 below. To compute \(\mathcal{L}\{f(t)\},\) we express it as an infinite series of integrals as follows:
    \begin{equation*} \begin{split} \mathcal{L}\{f(t)\}& =\int_0^\infty f(t) e^{-st}\, dt\\ &= \int_0^T f(t) e^{-st}\, dt+ \int_T^{2T} f(t) e^{-st}\, dt+ \int_{2T}^{3T} f(t) e^{-st}\, dt +\dots\\ &=\int_0^T f(t) e^{-st}\, dt + \int_0^T f(t+T) e^{-s(t+T)}\, dt\\ & \quad + \int_0^T f(t+2T) e^{-s(t+2T)}\, dt+ \dots+ \int_0^T f(t+nT) e^{-s(t+nT)}\, dt+\cdots\\ &=\int_0^T f(t) e^{-st}\, dt\;\Big(1+e^{-sT}+ e^{-2sT} + \dots+ e^{-nsT}+\cdots \Big)\\ &=\int_0^T f(t) e^{-st}\, dt\;\; \sum_{n=0}^\infty e^{-nsT}. \end{split} \end{equation*}
    Since the geometric series \(\displaystyle \sum_{n=0}^\infty e^{-nsT}\) converges for \(s>0\) to \(\displaystyle \frac{1}{1- e^{-sT}},\) we obtain
    \begin{equation} \boxed{\mathcal{L}\{f(t)\} = \frac{1}{1- e^{-sT}}\int_0^T f(t) e^{-st}\, dt.}\tag{5.4.1} \end{equation}
    🔗

    Example 5.4.3.

    Let us revisit \(\mathcal{L}\{\sin(\omega t)\}.\)
    🔗
    🔗
    Using (5.4.1) with \(T= 2\pi/\omega\text{,}\) we obtain
    \begin{equation*} \mathcal{L}\{\sin(\omega t)\} = \frac{1}{1- e^{-2\pi s/\omega}}\int_0^{2\pi/\omega} \sin(\omega t) e^{-st}\, dt. \end{equation*}
    We compute
    \begin{equation*} \int_0^{2\pi/\omega} \sin(\omega t) e^{-st}\, dt=\frac{\omega}{s^2+\omega^2}\left( 1- e^{-2\pi s/\omega}\right). \end{equation*}
    Hence
    \begin{equation*} \mathcal{L}\{\sin(\omega t)\} =\frac{\omega}{s^2+\omega^2}, \end{equation*}
    as expected.
    🔗
    The example above suggests that the direct computation of the Laplace transform of a periodic function may be more convenient.
    🔗
    The example below concerns a periodic function acting as the source term of an initial value problem.
    🔗

    Example 5.4.4.

    Consider the initial value problem \(y'+ y = f(t), \; y(0) = 0,\) where \(f(t)\) is given by
    \begin{equation*} f(t) = \begin{cases} 1 &\text{if } 0\le t\le 1,\\ 0 &\text{if } 1< t < 2, \end{cases} \end{equation*}
    and \(f(t) = f(t+2)\) for all \(t\) in \((-\infty, \infty).\)
    🔗
    🔗
    The graph of \(f\) on \([0, \infty)\) is shown below.
    🔗
    Then
    \begin{equation*} \begin{split} F(s) &= \frac{1}{1-e^{-2s}}\int_0^2 f(t) e^{-st}\, dt\\ &= \frac{1}{1-e^{-2s}}\int_0^1 e^{-st}\, dt\\ & = \frac{1}{1-e^{-2s}}\left(\frac{e^{-s}-1}{-s}\right)\\ &= \frac{1}{s(1+e^{-s})}. \end{split} \end{equation*}
    Taking the Laplace transform of the differential equation \(y' + y = f(t) \text{,}\) we get
    \begin{equation*} sY(s) - y(0) + Y(s) = F(s). \end{equation*}
    With \(y(0) = 0,\) we have
    \begin{equation*} Y(s) = \frac{1}{s(s+1)(1+e^{-s})}. \end{equation*}
    Since
    \begin{equation*} \displaystyle \frac{1}{s(s+1)} = \frac{1}{s}- \frac{1}{s+1} \end{equation*}
    and
    \begin{equation*} \displaystyle\frac{1}{1+e^{-s}} = \sum_{n=0}^\infty (-1)^ne^{-ns}, \end{equation*}
    it follows that
    \begin{equation*} Y(s) = \left(\frac{1}{s}- \frac{1}{s+1}\right)\sum_{n=0}^\infty (-1)^ne^{-ns}. \end{equation*}
    In view of the shifting on the \(t-\)axis property discussed in Example 5.1.13, the solution \(y(t)\) of the initial value problem is
    \begin{equation} y(t) = \sum_{n=0}^{\infty}(-1)^n \mathcal{U}(t-n)\; (1- e^{-(t-n)}).\tag{5.4.2} \end{equation}
    The graph of this solution is in Figure 5.4.6 below. The solution \(y(t)\) is a piecewise continuous periodic function with period 2, and it converges to the periodic function \(f(t)\) as \(t\to\infty.\)
