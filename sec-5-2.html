    Section 5.2 Applications to Initial Value Problems

    As point out earlier in Section 5.1, we will need to convert a linear initial value problem into an algebraic equation using the Laplace transform. After solving the equation in the \(s-\)domain for the Laplace transform of the solution function, we then apply the inverse Laplace transform operator \(\mathscr{L}^{-1}\) to return to the \(t-\)domain and obtain the solution of the initial value problem. For these steps, we will need the Laplace transform of derivatives.
    🔗
    Suppose that \(y(t)\) is continuous on \([0,\infty)\) and of exponential order as \(t\to\infty.\) Then there exists constants \(\alpha\text{,}\) \(M>0\) and \(t_0>0\) such that
    \begin{equation*} |y(t)| \le M e^{\alpha t}\quad \text{for all } t\ge t_0. \end{equation*}
    The number \(\alpha\) is referred to as an exponential order of \(y\text{.}\) Then, for \(s> \alpha,\) we have
    \begin{equation*} |y(t) e^{-st}|\le M e^{-(s-\alpha)t} \quad \text{for all }t\ge t_0, \end{equation*}
    and consequently,
    \begin{equation*} \lim\limits_{t\to\infty}y(t) e^{-st} =0\quad \text{for }s>\alpha. \end{equation*}
    Suppose, further, that \(y'\) is piecewise continuous on \([0,\infty).\) Then, using integration by parts, we have
    \begin{align*} \mathscr{L}\{y'(t)\} \amp = \int_0^{\infty} e^{-st} y'(t) \, dt \\ \amp = \left[ e^{-st} y(t) \right]_0^{\infty} + s \int_0^{\infty} e^{-st} y(t) \, dt\\ \amp = -y(0) + s\mathscr{L}\{y(t)\}\\ \amp = sY(s) - y(0) \end{align*}
    for \(s>\alpha.\) Thus,
    \begin{equation} \boxed{ \mathscr{L}\{y'(t)\} = sY(s) - y(0).}\tag{5.2.1} \end{equation}
    🔗
    We extend the above idea to higher order derivatives. For instance, suppose that \(y(t)\) and \(y'(t)\) are continuous and of exponential order. If \(y''(t)\) exists and is piecewise continuous on \([0, \infty),\) then applying the formula ((5.2.1)) first to \(z'(t)\) with \(z(t) = y'(t)\) and then to \(y'(t)\text{,}\) we obtain
    \begin{align*} \mathscr{L}\{y''(t)\}\amp =\mathscr{L}\{z'(t)\} \\ \amp = sZ(s) - z(0) \\ \amp= s\mathscr{L}\{y'(t)\}- y'(0) \\ \amp = s(sY(s) - y(0)) - y'(0) \\ \amp =s^2Y(s) - sy(0)-y'(0), \end{align*}
    i.e.,
    \begin{equation} \boxed{\mathscr{L}\{y''(t)\} = s^2Y(s) - sy(0)-y'(0).}\tag{5.2.2} \end{equation}
    🔗
    Applying these ideas repeatedly to higher order derivatives, we obtain the following theorem whose proof is omitted.
    🔗

    Theorem 5.2.1. (Laplace transform of derivatives).

    Let \(n\ge 1\) be a positive integer. If \(y, y', \ldots, y^{(n-1)}\) are continuous on \([0, \infty)\) and of exponential order as \(t\to\infty\) and if \(y^{(n)}(t)\) is piecewise continuous on \([0, \infty)\) and of exponential order as \(t\to\infty,\) then
    \begin{equation*} \mathscr{L}\{y^{(n)}(t)\} = s^n Y(s) - s^{n-1}y(0) - s^{n-2}y'(0) - \cdots - y^{(n-1)}(0), \end{equation*}
    where \(Y(s) = \mathscr{L}\{y(t)\}\text{.}\)
    🔗
    🔗
    Theorem 5.2.1 presents how \(y(0),y'(0), \dots, y^{(n-1)}(0)\) naturally arise, reflecting the initial conditions of an \(n\)-th order linear initial value problem and indicating the potential use of the Laplace transform to covert a linear differential equation with constant coefficients to an algebraic equation with no derivatives of the Laplace transform of the solution variable. Specifically, consider the linear initial value problem of the form:
    \begin{equation} \begin{cases} &a_n y^{(n)} + a_{n-1}y^{(n-1)}+ \cdots+a_1y'+a_0 y = f(t),\\ &y(0) = y_0, \dots, y^{(n-1)}(0)= y_{n-1}. \end{cases}\tag{5.2.3} \end{equation}
    Applying Theorem 5.2.1 in conjunction with the linearity of \(\mathscr{L}\text{,}\) the initial value problem in (5.2.3) gives rise to the following algebraic equation in the \(s-\)domain:
    \begin{equation} \tag{5.2.4} \end{equation}
    which takes the form
    \begin{equation} p(s)Y(s)= q(s) +F(s),\tag{5.2.5} \end{equation}
    where \(p(s) = a_ns^n+a_{n-1}s^{n-1} + \cdots+ a_1s+a_0\) and \(q(s)\) is a polynomial in \(s\) obtained from all the remaining terms on the left side of the equation (5.2.5). It then follows that
    \begin{equation} Y(s)= \frac{q(s)}{p(s)} +\frac{F(s)}{p(s)}.\tag{5.2.6} \end{equation}
    The solution \(y(t)\) to the initial value problem ((5.2.3)) is given by
    \begin{equation} y(t) = \mathscr{L}^{-1}\{Y(s)\} =\mathscr{L}^{-1}\left\{\frac{q(s)}{p(s)}\right\} +\mathscr{L}^{-1}\left\{\frac{F(s)}{p(s)}\right\}.\tag{5.2.7} \end{equation}
    To find
    \begin{equation*} \mathscr{L}^{-1}\left\{\frac{q(s)}{p(s)}\right\} \quad\text{and}\quad\mathscr{L}^{-1}\left\{\frac{F(s)}{p(s)}\right\}, \end{equation*}
    we typically use partial fraction decomposition of \({q(s)}/{p(s)}\) and \({F(s)}/{p(s)}\) and then use appropriate inverse Laplace transforms term by term. For more details on the techniques of partial fraction decomposition, see Appendix Section 6.2.
    🔗
    In principle, one should verify that the equation (5.2.7) satisfies the initial value problem (5.2.3) by verifying that \(y(t)\) solves the differential equation and satisfies the initial conditions, especially important when the differential equation is defined also for \(t < 0\) since the Laplace transform only makes sense to functions on \([0, \infty)\text{.}\) However, this verification is unnecessary when \(f(t)\) is continuous on an open interval containing \(t = 0\text{,}\) as the initial value problem (5.2.3) is guaranteed to have a unique local solution.
    🔗
    In the following examples, we apply the Laplace transform method to solve initial value problems.
    🔗

    Example 5.2.2.

    \begin{equation*} y'-y= 2\cos(3t), \quad y(0) = 1. \end{equation*}
    Answer.
    \(y(t)=\frac{6}{5}e^t-\frac{1}{5}\cos(3t)+\frac{3}{5}\sin(3t)\) on \((-\infty, \infty)\text{.}\)
    🔗
    🔗
    Solution.
    Applying the Laplace transform to both sides of the equation, we have
    \begin{equation*} \begin{aligned} \mathscr{L}\{y'\} - \mathscr{L}\{y\} &= 2\mathscr{L}\{\cos(3t)\} \\ (sY(s) - y(0)) - Y(s) &= 2\cdot \frac{s}{s^2 + 9} \\ (s-1)Y(s) &= 1 + \frac{2s}{s^2 + 9} \\ Y(s) &= \frac{1}{s-1} + \frac{2s}{(s-1)(s^2 + 9)}. \end{aligned} \end{equation*}
    To find the inverse Laplace transform of \({2s}/{((s-1)(s^2 + 9))}\text{,}\) we use partial fraction decomposition to write
    \begin{equation*} \frac{2s}{(s-1)(s^2 + 9)} = \frac{A}{s-1} + \frac{Bs + C}{s^2 + 9}. \end{equation*}
    Solving for \(A, B, C\text{,}\) we have \(A = 1/5, B = -1/5, C = 9/5.\) Thus,
    \begin{equation*} \begin{aligned} Y(s) &= \frac{1}{s-1} + \frac{1/5}{s-1} + \frac{-1/5 s + 9 /5}{s^2 + 9} \\ &= \frac{6/5}{s-1} + \frac{-1/5 s + 9/5}{s^2 + 9}. \end{aligned} \end{equation*}
    Applying the inverse Laplace transform term by term, we have
    \begin{equation*} \begin{aligned}y(t) &= \mathscr{L}^{-1}\left\{\frac{6/5}{s-1}\right\} + \mathscr{L}^{-1}\left\{\frac{-1/5 s + 9/5}{s^2 + 9}\right\} \\ &= \frac{6}{5}e^t - \frac{1}{5}\cos(3t) + \frac{3}{5}\sin(3t). \end{aligned} \end{equation*}
    🔗
    🔗
    🔗

    Example 5.2.3.

    Solve the initial value problem using the Laplace transform:
    \begin{equation*} y''-5y'+4y= t, \quad y(0) = 1, y'(0) = 0. \end{equation*}
    🔗
    Answer.
    \(y(t)=e^t-\dfrac{5}{16}e^{4t}+\dfrac14 t+\dfrac{5}{16}\) on \((-\infty, \infty)\text{.}\)
    🔗
    🔗
    Solution.
    \begin{equation*} \begin{aligned}&\mathscr{L}\{y''\} - 5\mathscr{L}\{y'\} + 4\mathscr{L}\{y\} = \mathscr{L}\{t\} \\ &(s^2 Y(s) - sy(0)-y'(0)) - 5(sY(s) - y(0)) + 4Y(s) = \frac{1}{s^2} \\ &(s^2 - 5s + 4)Y(s) = \frac{1}{s^2} + s - 5 \\ &Y(s) = \frac{1}{s^2(s-1)(s-4)} + \frac{s-5}{(s-1)(s-4)}=\frac{s^3-5s^2+1}{s^2(s-1)(s-4)}. \end{aligned} \end{equation*}
    Decompose into partial fractions:
    🔗
    \begin{equation*} \frac{s^3-5s^2+1}{s^2(s-1)(s-4)} =\frac{A}{s}+\frac{B}{s^2}+\frac{C}{s-1}+\frac{D}{s-4}. \end{equation*}
    Solving for the constants gives
    🔗
    \begin{equation*} A=\frac{5}{16},\qquad B=\frac{1}{4},\qquad C=1,\qquad D=-\frac{5}{16}. \end{equation*}
    Hence
    🔗
    \begin{equation*} Y(s)=\frac{5}{16s}+\frac{1}{4s^2}+\frac{1}{s-1}-\frac{5}{16(s-4)}. \end{equation*}
    Now apply the inverse Laplace transform:
    🔗
    Therefore,
    \begin{equation*} \begin{aligned} y(t)\amp =\mathscr{L}^{-1}\left\{\frac{5}{16s}\right\}+ \mathscr{L}^{-1}\left\{\frac{1}{4s^2}\right\}\\ \amp +\mathscr{L}^{-1}\left\{\frac{1}{s-1}\right\} -\mathscr{L}^{-1}\left\{\frac{5}{16(s-4)}\right\}\\ \amp =\frac{5}{16}+\frac{1}{4}t+e^t-\frac{5}{16}e^{4t}. \end{aligned} \end{equation*}
    We can show that
    \begin{equation*} y(t)=\frac{5}{16}+\frac{1}{4}t+e^t-\frac{5}{16}e^{4t} \quad \text{ on } (-\infty, \infty) \end{equation*}
    is the solution of the initial value problem on by verifying that it satisfies the differential equation and the initial conditions.
    🔗
    🔗

    Example 5.2.4. Move this to a later section.

    \begin{equation*} y'' + 4y' + 5y = e^{-t}, \quad y(0) = 0, \; y'(0) = 2. \end{equation*}
    Solution.
    Applying the Laplace transform to both sides of the equation, we have
    \begin{equation*} \begin{aligned} &\mathscr{L}\{y''\} + 4\mathscr{L}\{y'\} + 5\mathscr{L}\{y\} = \mathscr{L}\{e^{-t}\} \\ &(s^2 Y(s) - sy(0) - y'(0)) + 4 (sY(s) - y(0)) + 5Y(s) = \frac{1}{s+1} \\ &(s^2 + 4s + 5)Y(s) = \frac{1}{s+1} + 2 \\ &Y(s) = \frac{1}{(s+1)(s^2 + 4s + 5)} + \frac{2}{s^2 + 4s + 5}. \end{aligned} \end{equation*}
    To find the inverse Laplace transform of \({1}/{((s+1)(s^2 + 4s + 5))}\text{,}\) we use partial fraction decomposition to write
    \begin{equation*} \frac{1}{(s+1)(s^2 + 4s + 5)} = \frac{A}{s+1} + \frac{Bs + C}{s^2 + 4s + 5}. \end{equation*}
    Solving for \(A, B, C\text{,}\) we have \(A = 1/5, B = -1/5, C = 1/5.\) Thus,
    \begin{equation*} \begin{aligned}Y(s) &= \frac{1/5}{s+1} + \frac{-1/5 s + 1/5}{s^2 + 4s + 5} + \frac{2}{s^2 + 4s + 5} \\ &= \frac{1/5}{s+1} + \frac{-1/5 s + 11/5}{s^2 + 4s + 5}. \end{aligned} \end{equation*}
    Applying the inverse Laplace transform term by term, we have
    \begin{equation*} \begin{aligned}y(t) &= \mathscr{L}^{-1}\left\{\frac{1/5}{s+1}\right\} + \mathscr{L}^{-1}\left\{\frac{-1/5 s + 11/5}{s^2 + 4s + 5}\right\} \\ &= \frac{1}{5}e^{-t} - \frac{1}{5}e^{-2t}\cos(t) + \frac{9}{5}e^{-2t}\sin(t). \end{aligned} \end{equation*}
    🔗
    🔗
    🔗
    The general principle for solving initial value problems by the Laplace transform method remains the same as demonstrated in the previous examples. The major effort to make is in computing the Laplace transforms of more complicated functions and finding inverse transforms of more complicated expressions. In the followings sections, we continue exploring additional methods for computing Laplace transforms and solve initial value problems that involve more complicated functions.
