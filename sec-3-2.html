    Section 3.2 Second Order Homogeneous Linear Differential Equations

    Objectives

    • Understand methods of solutions for second order homogeneous linear differential equations
      🔗
    🔗
    We now discuss the Wronskian of solutions of second order homogeneous linear equations.
    🔗
    Let \(y_1\) and \(y_2\) be solutions of the homogeneous linear differential equation
    \begin{equation*} y'' + a_1(x)y' + a_0(x)y = 0, \end{equation*}
    where \(a_1(x)\) and \(a_0(x)\) are continuous functions on an interval \(I.\) The Wronskian of \(y_1\) and \(y_2\text{,}\) denoted by \(W(y_1, y_2)(x)\) , \(W(y_1(x), y_2(x))\text{,}\) or simply \(W(x)\text{,}\) is defined as
    \begin{equation*} W(x) = \begin{vmatrix} y_1(x) & y_2(x) \\ y_1'(x) & y_2'(x) \end{vmatrix} = y_1(x)y_2'(x) - y_1'(x)y_2(x). \end{equation*}
    Let us examine the derivative of the Wronskian \(W(x).\) We have
    \begin{equation*} \begin{split} W'(x) &= y_1(x)y_2''(x)+y'_1(x)y'_2(x) - y_1'(x)y'_2(x)- y_2(x)y''_1(x)\\ &=y_1(x)y_2''(x)-y_2(x)y''_1(x)\\ &=y_1(x)\big(-a_1(x) y'_2(x)- a_0(x)y_2(x)\big)\\ &\quad -y_2(x)\big(-a_1(x) y'_1(x)- a_0(x)y_1(x)\big)\\ &=- a_1(x) W(x), \end{split} \end{equation*}
    where we used \(y_1''(x) = -a_1(x) y'_1(x)- a_0(x)y_1(x)\) and \(y_2''(x) = -a_1(x) y'_2(x)- a_0(x)y_2(x).\) It then follows that
    \begin{equation} W(x) =W(x_0)\; e^{-\int_{x_0}^x a_1(t)\, dt},\tag{3.2.1} \end{equation}
    where \(x_0\) is a fixed number in \(I.\) The formula (3.2.1) is known as Abel’s formula. It follows from this formula that \(W(x)\) is a constant when \(a_1(x) =0\) on \(I.\) Also, if \(W(x_0)= 0\text{,}\) then \(W(x) = 0\) for all \(x\) in \(I.\) On the other hand, since
    \begin{equation*} e^{-\int_{x_0}^x a_1(t)\, dt} >0 \quad \mbox{ for all } x \mbox{ in } I, \end{equation*}
    it follows that \(W(x) <0\) for all \(x\) in \(I\) if \(W(x_0)<0,\) and \(W(x) >0\) for all \(x\) in \(I\) if \(W(x_0)>0.\) Thus, either \(W(x)\) is identically zero or it does not change its sign in \(I.\)
    🔗
    The analysis of the Wronskian together with the formula (3.2.1) above also hold for higher order homogeneous linear equations (see Section?? Higher Order).
    🔗

    Theorem 3.2.1. (Criterion for Linearly Indepedent Solutions).

    Two solutions \(y_1\) and \(y_2\) of
    \begin{equation} y'' + a_1(x)y' + a_0(x)y = 0,\tag{3.2.2} \end{equation}
    where \(a_1(x)\) and \(a_0(x)\) are continuous functions on an interval \(I,\) are linearly independent on \(I\) if and only if \(W(y_1, y_2)(x)\ne 0\) for all \(x\) in \(I.\)
    🔗
    🔗

    Proof.

    Let \(y_1\) and \(y_2\) be linearly independent solutions of (3.2.2) on \(I\text{.}\) Suppose, for a contradiction, that there is a point \(x_0\) in \(I\) such that \(W(y_1,y_2)(x_0)= 0.\) Let \(c_1\) and \(c_2\) be constants. Then the linear system
    \begin{equation} \begin{cases} c_1 y_1(x_0) + c_2 y_2(x_0) = 0\\ c_1 y_1'(x_0) + c_2 y_2'(x_0) = 0 \end{cases}\tag{3.2.3} \end{equation}
    has a solution \(c_1, c_2,\) not both zero because the determinant of the coefficient matrix is zero. Fix \(c_1, c_2\text{,}\) not both zero, satisfying (3.2.3), and define the function
    \begin{equation*} \phi(x) = c_1 y_1(x) + c_2 y_2(x) \end{equation*}
    on \(I.\) By (3.2.3) and the superposition principle, \(y = \phi(x)\) is a solution on \(I\) of the initial value problem
    \begin{equation*} y''+a_1(x) y' + a_0(x) y = 0, \quad y(x_0) = 0,\; y'(x_0) =0. \end{equation*}
    The uniqueness part of Theorem Theorem 3.1.3 implies \(\phi(x) =0\) on \(I,\) that is,
    \begin{equation*} c_1 y_1(x) + c_2 y_2(x) = 0 \end{equation*}
    on \(I.\) Thus, \(y_1\) and \(y_2\) are linearly dependent on \(I\text{,}\) contradicting our assumption. Hence \(W(y_1,y_2)(x)\neq 0\) for every \(x\) in \(I.\)
    🔗
    Suppose that \(W(y_1, y_2)(x)\ne 0\) on \(I.\) By Theorem 3.2.1, \(y_1\) and \(y_2\) are linearly independent on \(I\text{.}\) Note that \(y_1\) and \(y_2\) need not be solutions (3.2.2) for this part of the theorem to hold.
    🔗
    🔗

    Subsection 3.2.1 Existence of Fundamental Sets of Solutions

    Definition 3.2.2.

    Consider the second order homogeneous linear differential equation
    \begin{equation*} y'' + a_1(x)y' + a_0(x)y = 0, \end{equation*}
    where \(a_1(x)\) and \(a_0(x)\) are continuous functions on an interval \(I.\) Two solutions \(y_1\) and \(y_2\) that are linearly independent on \(I\) are said to form a fundamental set of solutions of the differential equation. Each of these solutions, \(y_1\) and \(y_2\text{,}\) is referred to as a fundamental solution.
    🔗
    🔗
    An important question arises: does a fundamental set of solutions always exist? The following theorem ensures the existence of a fundamental set of solutions for second order homogeneous linear equations when \(a_1(x)\) and \(a_0(x)\) are continuous. These concepts extend naturally to higher order equations.
    🔗

    Theorem 3.2.3.

    Consider the second order homogeneous linear differential equation
    \begin{equation} y'' + a_1(x)y' + a_0(x)y = 0,\tag{3.2.4} \end{equation}
    where \(a_1(x)\) and \(a_0(x)\) are continuous functions on an interval \(I.\) Then there exist two functions \(y_1\) and \(y_2\) on \(I\) that form a fundamental set of solutions for (3.2.4).
    🔗
    🔗

    Proof.

    By Theorem 3.1.3, there exists a unique solutions of \(y_1\) the initial value problem
    \begin{equation*} y'' + a_1(x)y' + a_0(x)y = 0, \quad y(x_0) = 1,\; y'(x_0) = 0, \end{equation*}
    and a unique \(y_2\) of the initial value problem
    \begin{equation*} y'' + a_1(x)y' + a_0(x)y = 0, \quad y(x_0) = 0,\; y'(x_0) = 1. \end{equation*}
    The two solutions \(y_1\) and \(y_2\) are linearly independent on \(I.\) In fact, let \(c_1, c_2\) be constants such that \(c_1y_1(x) + c_2y_2( x) = 0\) for all \(x\) in \(I.\) Since \(x_0\) is in \(I,\) we have \(c_1y_1(x_0) + c_2y_2( x_0) = 0,\) which implies \(c_1 =0\text{.}\) We then only have \(c_2y_2(x) = 0\) for all \(x\) in \(I.\) Differentiating \(c_2y_2( x) = 0\) yields \(c_2 y'_2(x) = 0\) for all \(x\) in \(I.\) Since \(y'_2(x_0) = 1,\) we must have \(c_2 = 0.\) Thus, \(\{y_1, y_2\}\) forms a fundamental set of solutions for (3.2.4) on \(I.\)
    🔗
    Let \(\{y_1, y_2\}\) be a fundamental set of solutions of (3.2.4) on the interval \(I,\) where both \(a_0(x)\) and \(a_1(x)\) are continuous. Denote by \(W(x)\) the Wronskian of \(y_1\) and \(y_2\text{.}\) Let \(x_0\) be a point in \(I.\) Since \(y_1\) and \(y_2\) are linearly independent on \(I,\) we have \(W(x_0)\ne 0.\) Then the Wronskian \(\widetilde{W}(x)\) of \(\frac{1}{ W(x_0)}y_1\) and \(y_2\) is
    \begin{equation*} \begin{split} \widetilde{W}(x) &=\frac{y_1(x) }{W(x_0)} y'_2(x) -\frac{y'_1(x) }{W(x_0)} y_2(x)\\ & = \frac{W(x)}{W(x_0)}. \end{split} \end{equation*}
    Obviously, \(\widetilde{W}(x_0)=1.\) Hence we have
    \begin{equation*} \widetilde{W}(x) = \widetilde{W}(x_0) e^{-\int_{x_0}^x a_1(t)\, dt} =e^{-\int_{x_0}^x a_1(t)\, dt}. \end{equation*}
    Thus, the solutions in a fundamental set can be scaled so that the Wronskian of the scaled solutions is
    \begin{equation*} e^{-\int_{x_0}^x a_1(t)\, dt}, \end{equation*}
    where \(x_0\) is a fixed point in \(I.\)
    🔗
    🔗

    Example 3.2.4.

    Find the Wronskian of a fundamental set of solutions of each of the following differential equations.
    🔗
    1. \(\displaystyle y''+\frac{2}{x}y'+\frac{10}{x^2}y = 0\)
      🔗
      🔗
    2. \(\displaystyle y''+\ln x\, y'+ 10 y =0\)
      🔗
      🔗
    3. \(\displaystyle y''-2xy'+5y = 0\)
      🔗
      🔗
    🔗
    Solution.
    (i) Take \(x_0= 1\) in \((0, \infty).\) There is a fundamental set of solutions whose Wronskian is
    \begin{equation*} e^{-\int_1^x a_1(t)\, dt} = e^{-\int_1^x \frac{2}{t}\, dt} = e^{-2 \ln x}= \frac{1}{x^2}. \end{equation*}
    (ii) Take \(x_0= 1\) in \((0, \infty).\) There is a fundamental set of solutions whose Wronskian is
    \begin{equation*} e^{-\int_1^x a_1(t)\, dt} =e^{-\int_1^x \ln t\, dt} = e^{x-x\ln x-1}. \end{equation*}
    (iii) Take \(x_0= 0\) in \((-\infty, \infty).\) There is a fundamental set of solutions whose Wronskian is
    \begin{equation*} e^{-\int_0^x a_1(t)\, dt} =e^{-\int_0^x (-2 t)\, dt} =e^{x^2}. \end{equation*}
    🔗
    🔗
    🔗
    The next theorem provides a method for finding the general solutions of second order homogeneous linear differential equations.
    🔗

    Theorem 3.2.5.

    Let \(\{y_1,y_2 \}\) form a fundamental set of solutions of the second order homogeneous linear differential equation
    \begin{equation} y'' + a_1(x)y' + a_0(x)y = 0,\tag{3.2.5} \end{equation}
    where \(a_1(x)\) and \(a_0(x)\) are continuous functions on an interval \(I.\) Then the functions of the form
    \begin{equation} y = c_1 y_1(x) + c_2 y_2(x)\tag{3.2.6} \end{equation}
    are solutions on \(I\) for any constants \(c_1\) and \(c_2\text{.}\) Moreover, every solution of (3.2.5) is of form (3.2.6) for some constants \(c_1\) and \(c_2\text{.}\)
    🔗
    🔗

    Proof.

    For any \(c_1\) and \(c_2\) constants, the Theorem 3.1.5 implies that
    \begin{equation} y = c_1 y_1(x) + c_2 y_2(x)\tag{3.2.7} \end{equation}
    is a solution of (3.2.5) on \(I.\)
    🔗
    To show that every solution of (3.2.5) is of this form (3.2.6), let \(\phi(x)\) be a solution of (3.2.5) on \(I\) and \(x_0\) be in \(I.\) Denote by \(a\) and \(b\) the constants \(\phi(x_0)\) and \(\phi'(x_0)\text{,}\) respectively. Then \(y = \phi(x)\) solves the initial value problem
    \begin{equation} y'' + a_1(x)y' + a_0(x)y = 0, \quad y(x_0) = a, y'(x_0) =b.\tag{3.2.8} \end{equation}
    On the other hand, let \(c_1\) and \(c_2\) be constants to be determined so that \(y(x) = c_1y_1(x)+ c_2y_2(x)\) satisfies \(y(x_0) = a, y'(x_0) = b.\) Then
    \begin{equation*} \begin{cases} c_1y_1(x_0) +c_2y_2(x_0) = a,\\ c_1y'_1(x_0) +c_2y'_2(x_0) = b.\\ \end{cases} \end{equation*}
    Since \(\{y_1, y_2\}\) is a linearly independent set of solutions, Theorem 3.2.1 implies \(W(y_1, y_2)(x_0)\ne 0,\) and therefore, by the Cramer’s rule, we find the unique values of \(c_1\) and \(c_2\) given by
    \begin{equation*} c_1 = \frac{ \begin{vmatrix} a & y_2(x_0) \\ b& y_2'(x_0) \end{vmatrix}}{W(y_1, y_2)(x_0)} \mbox{ and } c_2 = \frac{ \begin{vmatrix} y_1(x_0)&a \\ y_1'(x_0)&b \end{vmatrix}}{W(y_1, y_2)(x_0)}. \end{equation*}
    With these values of \(c_1\) and \(c_2\text{,}\) the function \(y= c_1 y_1 + c_2 y_2\) solves the same initial value problem (3.2.8) as \(\phi(x)\text{.}\) By the uniqueness part of Theorem Theorem 3.1.3, we conclude that \(\phi(x) = c_1 y_1(x) + c_2 y_2(x)\) on \(I.\)
    🔗
    🔗
    🔗

    Subsection 3.2.2 Constructing Fundamental Sets of Solutions

    We now discuss methods for determining fundamental sets of solutions for second order homogeneous linear differential equations with constant coefficients, specifically
    \begin{equation} ay''+by'+cy =0,\tag{3.2.9} \end{equation}
    where \(a, b, c\) are constant coefficients and \(a\ne 0.\) This equation can equivalently be written in the standard form
    \begin{equation} y''+\frac{b}{a}y'+\frac{c}{a}y =0.\tag{3.2.10} \end{equation}
    Because the coefficient functions are constants and defined on \((-\infty, \infty),\) the interval \(I\) in the existence and uniqueness theorem (Theorem 3.1.3) for (3.2.9) is the entire set of real numbers \((-\infty, \infty).\) Consequently, the solutions in any fundamental set are defined on \((-\infty, \infty).\)
    🔗
    Motivated from first order linear equations, we expect that solutions appear as exponential functions. So, we let \(y = e^{\lambda x}\) be a trial solution of (3.2.9) for some number (possibly complex) \(\lambda.\) Differentiating \(y = e^{\lambda x}\) twice with respect to \(x\) and substituting \(y, y',y''\) into (3.2.9) yields
    \begin{equation*} (a\lambda^2+ b\lambda +c)e^{\lambda x} = 0 \end{equation*}
    for all \(x\) in \((-\infty, \infty).\) In particular, when \(x=0,\) we obtain
    \begin{equation} a\lambda^2+ b\lambda +c =0,\tag{3.2.11} \end{equation}
    which is called the auxiliary (or characteristic)
     1 
    The polynomial \(p(\lambda) = a\lambda^2+b\lambda+c\) is called the characteristic polynomial for the differential equation \(ay''+by'+cy=0\text{.}\)
    equation for (3.2.9). The roots
     2 
    Solutions would be a more precise term, but we use roots here to avoid confusion with solutions of differential equations.
    of (3.2.11) are given by
    \begin{equation} \lambda=\frac{-b\pm\sqrt{b^2-4ac}}{2a},\tag{3.2.12} \end{equation}
    which are distinct real, complex, or repeated according as \(b^2-4ac>0\text{,}\) \(b^2-4ac<0\text{,}\) or \(b^2-4ac=0\text{.}\) We now construct a fundamental set of solutions for each case.
    🔗

    Case I. \(b^2-4ac>0\) (Distinct Real Roots).

    When \(b^2-4ac>0\text{,}\) the auxiliary equation (3.2.11) possesses two distinct real roots \(\lambda_1\) and \(\lambda_2\) given by (3.2.12). Take
    \begin{equation} \lambda_1= -\frac{b}{2a}+ \frac{\sqrt{b^2-4ac}}{2a} \quad\mbox{and}\quad \lambda_2=-\frac{b}{2a}- \frac{\sqrt{b^2-4ac}}{2a}.\tag{3.2.13} \end{equation}
    Since \(\lambda_1\ne \lambda_2,\) it follows that \(y_1=e^{\lambda_1x}\) and \(y_2=e^{\lambda_2x}\) are linearly independent solutions of (3.2.9). We confirm the linear independence of \(y_1\) and \(y_2\) by evaluating their Wronskian:
    \begin{equation*} W(y_1,y_2)(x) = \begin{vmatrix} e^{\lambda_1x}&e^{\lambda_2x}\\ \lambda_1 e^{\lambda_1x}&\lambda_2e^{\lambda_2x} \end{vmatrix} = (\lambda_2-\lambda_1)e^{\lambda_1x} e^{\lambda_2x} \ne 0 \end{equation*}
    on \((-\infty, \infty)\text{.}\) Therefore, \(\{y_1, y_2\}\) is a fundamental set of solutions. By Theorem 3.2.5, the general solution of (3.2.9) is
    \begin{equation} \boxed{y = c_1e^{\lambda_1x}+c_2e^{\lambda_2x},}\tag{3.2.14} \end{equation}
    where \(c_1\) and \(c_2\) are arbitrary constants.
    🔗
    🔗

    Case II. \(b^2-4ac<0\) (Complex Roots).

    When \(b^2-4ac<0\text{,}\) we have \(\sqrt{b^2-4ac}= i\sqrt{4ac-b^2},\) where \(i = \sqrt{-1}\text{,}\) the imaginary unit in the complex number system. Then the auxiliary equation (3.2.11) possesses two distinct complex roots \(\lambda_1\) and \(\lambda_2\) given by (3.2.12). Take
    \begin{equation} \lambda_1= -\frac{b}{2a}+ i\frac{\sqrt{4ac-b^2}}{2a} \quad\mbox{and}\quad \lambda_2=-\frac{b}{2a}- i\frac{\sqrt{4ac-b^2}}{2a}.\tag{3.2.15} \end{equation}
    Let
    \begin{equation*} \alpha = -\frac{b}{2a} \quad \mbox{and}\quad \beta =\frac{\sqrt{4ac-b^2}}{2a}. \end{equation*}
    It is evident that \(\beta \ne 0.\) Then the roots \(\lambda_1=\alpha+ i\beta\) and \(\lambda_2=\alpha - i \beta\) of the auxiliary equation (3.2.11) give the complex-valued solutions \(z_1=e^{\lambda_1x}\) and \(z_2=e^{\lambda_2x}\) of (3.2.9). To extract real-valued solutions, we use first use the Euler formula
    \begin{equation*} e^{i\theta} = \cos \theta + i \sin \theta, \quad \theta \mbox{ in radian,} \end{equation*}
    and write
    \begin{equation*} \begin{split} z_1&=e^{\lambda_1x} = e^{(\alpha+ i\beta)x} = e^{\alpha x} e^{i\beta x} =e^{\alpha x} \cos(\beta x)+ ie^{\alpha x}\sin(\beta x), \mbox{and }\\ z_2&=e^{\lambda_1x} = e^{(\alpha- i\beta)x} = e^{\alpha x} e^{-i\beta x} =e^{\alpha x} \cos(\beta x)- ie^{\alpha x}\sin(\beta x). \end{split} \end{equation*}
    It is evident that
    \begin{equation*} z_1 + z_2 =2e^{\alpha x} \cos(\beta x) \quad\mbox{and}\quad z_1 - z_2 =2ie^{\alpha x} \sin(\beta x), \end{equation*}
    which imply
    \begin{equation*} \begin{split} e^{\alpha x} \cos(\beta x) &=\frac{1}{2}\left(z_1+z_2\right)= \frac{1}{2}z_1 + \frac{1}{2}z_2\\ e^{\alpha x} \sin(\beta x)&=\frac{1}{2i}\left(z_1-z_2\right)=\frac{1}{2i}z_1 - \frac{1}{2i}z_2. \end{split} \end{equation*}
    Since \(z_1\) and \(z_2\) are solutions of (3.2.10) (and equivalently of (3.2.9)), it follows from the superposition principle (Theorem 3.1.5) that \(e^{\alpha x} \cos(\beta x)\) and \(e^{\alpha x} \sin(\beta x)\) are solutions of the homogeneous equation (3.2.9). Denote \(y_1=e^{\alpha x} \cos(\beta x)\) and \(y_2=e^{\alpha x} \sin(\beta x).\) Then \(y_1\) and \(y_2\) are linearly independent solutions because their Wronskian satisfies
    \begin{equation*} \begin{split} W(y_1,y_2)(x) &= \begin{vmatrix} e^{\alpha x} \cos(\beta x)&e^{\alpha x} \sin(\beta x)\\ -\beta e^{\alpha x} \sin(\beta x) +\alpha e^{\alpha x} \cos(\beta x) &\beta e^{\alpha x} \cos(\beta x) +\alpha e^{\alpha x} \sin(\beta x) \end{vmatrix}\\ &=e^{2\alpha x} \beta\\ &\ne 0 \end{split} \end{equation*}
    for all \(x\) in \((-\infty, \infty)\text{.}\) Thus, \(\{y_1, y_2\}\) is a fundamental set of solutions. By Theorem 3.2.5, the general solution of (3.2.9) is
    \begin{equation} \boxed{y = c_1e^{\alpha x} \cos(\beta x)+c_2e^{\alpha x} \sin(\beta x) = e^{\alpha x} \big(c_1\cos(\beta x)+c_2 \sin(\beta x)\big),}\tag{3.2.16} \end{equation}
    where \(c_1\) and \(c_2\) are arbitrary constants.
    🔗
    🔗

    Case III. \(b^2-4ac=0\) (Repeated Root).

    In this case, the auxiliary equation (3.2.11) has a repeated real root
    \begin{equation*} \lambda_1 = \lambda_2 = -\frac{b}{2a}, \end{equation*}
    as given in (3.2.12). Consequently, we obtain only a single solution:
    \begin{equation*} y_1 = e^{\lambda_1 x} = e^{-\frac{b}{2a}x}. \end{equation*}
    Since a fundamental set of solutions for (3.2.9) requires two linearly independent solutions, we must determine a second solution \(y_2\) so that \(\{y_1, y_2\}\) forms a fundamental set of solutions.
    🔗
    We next discuss a method known as reduction of order, which allows us to find a second linearly independent solution of a second order homogeneous linear differential equation, even when the equation has variable coefficients, as in (3.2.17), given that one nontrivial solution is already known.
    🔗
    Let \(y_1(x)\) be a known solution of
    \begin{equation} y'' + a_1(x)y' + a_0(x)y = 0,\tag{3.2.17} \end{equation}
    where \(a_0(x)\) and \(a_1(x)\) are continuous functions on an interval \(I,\) satisfying either \(y_1(x)>0\) for all \(x\) in \(I\) or \(y_1(x) <0\) for all \(x\) in \(I\text{.}\) Theorem 3.1.3 guarantees the existence of such a nonzero solution on \(I\) for the initial condition \(y(x_0) = y_0 \neq 0.\) We assume \(y_1(x) > 0\) on \(I\text{;}\) the other case can be treated similarly.
    🔗
    Our goal is to find a second solution \(y_2\) such that \(\{y_1, y_2\}\) is linearly independent on \(I.\) To this end, let \(u(x)\) be a function on \(I\) such that \(y_2 (x) = u(x) y_1(x)\) is a solution of (3.2.17). If we find a non-constant function \(u,\) then it will follow that \(\{y_1, y_2\}\) is a fundamental set of solutions. Then, suppressing \(x\) from the arguments for simplicity, we have
    \begin{equation*} \begin{split} y'_2 &= u' y_1 + u y'_1\quad \mbox{ and }\\ y''_2 &= u'' y_1 +2u' y'_1+ u y''_1. \end{split} \end{equation*}
    Substituting these into (3.2.17) gives
    \begin{equation*} u'' y_1 +2u' y'_1+ u y''_1 +a_1u' y_1 + a_1u y'_1 + a_0 u y_1=0.\, \end{equation*}
    which becomes
    \begin{equation*} u'' y_1 +u'\big(2 y'_1+ a_1y_1\big) +u\big( y''_1 + a_1 y'_1+ a_0 y_1\big) =0. \end{equation*}
    Since \(y_1\) is a solution of (3.2.17), we have \(y''_1 + a_1 y'_1 +a_0 y_1 = 0,\) and therefore we obtain
    \begin{equation*} u'' y_1 +u'\Big(2 y'_1 +a_1y_1\Big) =0. \end{equation*}
    We observe that there is no \(u\) in this equation. Let \(v = u'.\) Then the equation becomes
    \begin{equation*} v' y_1 +v\Big(2 y'_1 +a_1y_1\Big) =0, \end{equation*}
    which is of the first order. Since \(y_1\) never vanishes on \(I\text{,}\) we can rewrite the above equation as
    \begin{equation} v' +\left(\frac{2 y'_1 +a_1y_1}{y_1}\right) v =0.\tag{3.2.18} \end{equation}
    A solution of (3.2.18) is given by
    \begin{equation*} v(x) = e^{-\int\left(2\frac{y'_1(x)}{y_1(x)}+a_1(x)\right)dx}= e^{-2\ln\lvert{y_1(x)}\rvert} \; e^{-\int a_1(x)\, dx}= \frac{e^{-\int a_1(x)\, dx}}{[y_1(x)]^2}. \end{equation*}
    Since \(v(x) = u'(x),\) we obtain
    \begin{equation*} u(x) = \int \frac{e^{-\int a_1(x)\, dx}}{[y_1(x)]^2}\, dx \end{equation*}
    on \(I,\) and consequently, the solution \(y_2(x)\) is given by
    \begin{equation} \boxed{y_2(x) = y_1(x) \int \frac{e^{-\int a_1(x)\, dx}}{[y_1(x)]^2}\, dx.}\tag{3.2.19} \end{equation}
    on \(I.\)
     3 
    This formula is due the French mathematician Joseph Liouville
    Thus, \(\{y_1, y_2\}\) is a fundamental set of solutions for (3.2.17) . We now return to finding a second linearly independent solution \(y_2\) for Case III. Since
    \begin{equation*} y_1 = e^{\lambda_1 x} = e^{-\frac{b}{2a}x} \end{equation*}
    and \(a_1(x) = \frac{b}{a}\) is a constant, the reduction of the order formula (3.2.19) gives
    \begin{equation*} y_2(x) = e^{-\frac{b}{2a}x} \int \frac{e^{-\int (b/a)\, dx}}{e^{-\frac{b}{a}x}}\, dx = e^{-\frac{b}{2a}x} \int 1\, dx = x e^{-\frac{b}{2a}x} \end{equation*}
    on \(I\text{.}\) Since we only need a second solution \(y_2\text{,}\) the constants of integration are set to zero for convenience. Thus, \(\{y_1, y_2\}\) forms a fundamental set of solutions. By Theorem 3.2.5, the general solution of (3.2.9) is
    \begin{equation} \boxed{ y = c_1 e^{-\frac{b}{2a}x} + c_2 x e^{-\frac{b}{2a}x} = (c_1 + c_2 x) e^{-\frac{b}{2a}x},}\tag{3.2.20} \end{equation}
    where \(c_1\) and \(c_2\) are arbitrary constants.
    🔗
    🔗
    An alternative method for finding a second linearly independent solution when the characteristic polynomial has a repeated root is outlined in Exercises.

    Remark 3.2.6.

    1. The method discussed above for finding \(u(x)\) is known as the reduction of order because the resulting differential equation (3.2.18) for \(v(x)\) is of first order; one order less than that of the original equation (3.2.17).
      🔗
      🔗
    2. If the known solution \(y_1\) vanishes at some points in the interval \(I\text{,}\) we can first apply the reduction of the order on a subinterval \(J\) of \(I\) where \(y_1\) does not change sign, and then extend the solution to all of \(I\) using a theorem on continuation of solutions (which is beyond the scope of this book). In practice, we may formally compute \(y_2(x)\) using (3.2.19) and then verify that \(y_2\) is actually well-defined on the entire interval \(I\text{.}\) For this, see Example 3.2.11.

    Example 3.2.7.

    Find the general solution of
    \begin{equation*} y''-16y = 0. \end{equation*}
    🔗
    Solution.
    Assume a solution of the form \(y = e^{\lambda x}\text{.}\) Then \(y' = \lambda e^{\lambda x}\) and \(y'' = \lambda^2 e^{\lambda x}\text{.}\) Substituting into the differential equation gives
    \begin{equation*} \lambda^2 e^{\lambda x} - 16 e^{\lambda x} = 0, \end{equation*}
    which gives
    \begin{equation*} (\lambda^2 - 16) e^{\lambda x} = 0. \end{equation*}
    Since \(e^{\lambda x} \neq 0\text{,}\) the auxiliary equation that we obtain is
    \begin{equation*} \lambda^2 - 16 = 0, \end{equation*}
    whose distinct real roots are
    \begin{equation*} \lambda_1 = 4\quad\mbox{and}\quad \lambda_2 =-4. \end{equation*}
    By (3.2.14), the general solution is
    \begin{equation*} y(x) = c_1 e^{4x} + c_2 e^{-4x}, \end{equation*}
    where \(c_1\) and \(c_2\) are arbitrary constants.

    Example 3.2.11.

    Given that \(y_1 = \sin x\) solves \(y'' + y = 0\) on \((-\infty, \infty)\text{,}\) find a second linearly independent solution using the formula (3.2.19) for the reduction of the order.
    🔗
    Solution.
    For \(y'' + y = 0\text{,}\) we have \(a_1 =0.\) Also, the given solution \(y_1 = \sin x\) vanishes at the points \(x = n\pi,\) \(n= 0, \pm 1, \pm 2, \dots.\) Therefore, we restrict \(y_1\) to an interval \(J\) not containing these points. For example, take \(J = (0, \pi).\) Applying the formula (3.2.19), a second solution \(y_2\) is given by
    \begin{equation*} \begin{split}y_2(x) &= \sin x \int \frac{1}{\sin^2x}\, dx\\ & = \sin x \int \csc^2 x\, dx= -\sin x \cot x\\ & = -\cos x\end{split} \end{equation*}
    on \(J.\) By scaling, we redefine \(y_2 = \cos x\) on \(J=(0, \pi).\) Since \(y_2= \cos x\) can be extended by the same formula to all of \((-\infty, \infty)\) and
    \begin{equation*} W(\cos x, \sin x) = 1\ne 0 \end{equation*}
    for all \(x\) in \((-\infty, \infty),\) it follows Theorem 3.2.1 that \(y_1= \sin x\) and \(y_2 = \cos x\) are linearly independent on \((-\infty, \infty).\)

    Techniques for finding fundamental sets of solutions for homogeneous linear equations with variable coefficients are discussed in Section 3.6. In this section, we will only discuss the Cauchy–Euler equations of the form
    \begin{equation} x^2y''+b xy' +cy =0,\quad x>0,\tag{3.2.21} \end{equation}
    where \(b\) and \(c\) are real constants. The method uses \(y = x^\lambda\) as a trial solution, with \(\lambda\) to be determined from the auxiliary equation
    \begin{equation*} \lambda^2 +(b-1)\lambda +c=0, \end{equation*}
    whose roots are real and distinct, complex, or repeated according as
    \begin{equation*} (b-1)^2-4c>0,\;\; (b-1)^2-4c<0,\;\; \mbox{ or } (b-1)^2-4c=0. \end{equation*}
    We next discuss three examples, one for each case.
    🔗

    Example 3.2.12.

    Find the general solution of the Cauchy-Euler equation
    \begin{equation*} x^2 y''+4xy'+y = 0,\quad x>0. \end{equation*}
    🔗
    Solution.
    Take \(y = x^\lambda\) as a trial solution, where \(\lambda\) is a number to be determined. Differentiating \(y = x^\lambda\) twice with respect to \(x\) and substituting \(y, y', y''\) into the differential equation, we obtain
    \begin{equation*} (\lambda^2+ 3\lambda +1)x^\lambda =0 \end{equation*}
    for all \(x\) in an interval \(I,\) so the auxiliary equation is
    \begin{equation*} \lambda^2+ 3\lambda +1=0 \end{equation*}
    with roots
    \begin{equation*} \lambda = \frac{-3\pm \sqrt{5}}{2}. \end{equation*}
    Let
    \begin{equation*} \lambda_1=\frac{-3+ \sqrt{5}}{2}\quad\mbox{and}\quad\lambda_2=\frac{-3- \sqrt{5}}{2}. \end{equation*}
    Obviously, \(\lambda_1<0\) and \(\lambda_2<0.\) From the two options \((-\infty,0)\) and \((0, \infty),\) let us take \(I = (0, \infty).\) Then \(y_1 = x^{\lambda_1}\) and \(\lambda_2 = x^{\lambda_2}\) are solutions on \(I,\) and their Wronskian
    \begin{equation*} \begin{split} W(y_1, y_2)(x) &= \begin{vmatrix} x^{\lambda_1}&x^{\lambda_2}\\ \lambda_1x^{\lambda_1-1}&\lambda_2x^{\lambda_2-1} \end{vmatrix} \\ &= (\lambda_2-\lambda_1) x^{(\lambda_1+\lambda_2-2)}\\ &=(\lambda_2-\lambda_1) x^{-5}\\ & \ne 0 \end{split} \end{equation*}
    for all \(x\) in \(I\) since \(\lambda_1+\lambda_2 = -3\text{.}\) Thus \(\{y_1, y_2\}\) is a fundamental set of solutions, and therefore the general solution of the Cauchy-Euler equation is
    \begin{equation*} y = c_1x^{\lambda_1}+c_2x^{\lambda_2} = c_1x^{\frac{-3+ \sqrt{5}}{2}}+c_2x^{\frac{-3- \sqrt{5}}{2}} \end{equation*}
    on \(I,\) where \(c_1\) and \(c_2\) are arbitrary constants.

    🔗
    🔗

    Example 3.2.13.

    Solve the Cauchy–Euler equation
    \begin{equation*} x^2 y'' - y' + 5y = 0, \quad x>0. \end{equation*}
    🔗
    Solution.
    Let \(y = x^\lambda\) be a trial solution. Then
    \begin{equation*} y' = \lambda x^{\lambda-1}, \qquad y'' = \lambda(\lambda-1)x^{\lambda-2}. \end{equation*}
    Substituting into the differential equation gives
    \begin{equation*} (\lambda^2 - 2\lambda + 5)x^\lambda = 0. \end{equation*}
    Then the auxiliary equation is
    \begin{equation*} \lambda^2 - 2\lambda + 5=0 \end{equation*}
    whose solutions are
    🔗
    \begin{equation*} \lambda = \frac{2 \pm \sqrt{-16}}{2} = 1 \pm 2i. \end{equation*}
    For complex numbers \(\lambda = 1 \pm 2i\text{,}\) we have
    \begin{equation*} x^{1 \pm 2i}= x\, x^{\pm 2i}= x e^{\pm 2i\ln x} = x\big(\cos(2 \ln x) \pm i \sin(2\ln x)\big). \end{equation*}
    Let \(y_1 = x\cos(2 \ln x)\) and \(y_2 = x\sin(2 \ln x).\) It is routine to verify that \(y_1\) and \(y_2\) are linearly independent solutions on \((0, \infty).\) Then the general solution of the Cauchy–Euler equation is
    \begin{equation*} y(x) = x \big(c_1 \cos(2 \ln x) + c_2 \sin(2 \ln x)\big), \quad x>0, \end{equation*}
    where \(c_1\) and \(c_2\) are arbitrary constants.
    🔗
    🔗
    🔗

    Example 3.2.14.

    Solve the Cauchy–Euler equation
    \begin{equation*} x^2 y'' - y' + y = 0, \quad x>0. \end{equation*}
    🔗
    Solution.
    Let \(y = x^\lambda\) be a trial solution. Then
    \begin{equation*} y' = \lambda x^{\lambda-1}, \qquad y'' = \lambda(\lambda-1)x^{\lambda-2}. \end{equation*}
    Substituting into the differential equation yields
    \begin{equation*} (\lambda^2 - 2\lambda + 1)x^\lambda = 0. \end{equation*}
    Then the auxiliary equation is
    \begin{equation*} \lambda^2 - 2\lambda + 1=0,\quad \mbox{i.e.},\; (\lambda-1)^2 = 0. \end{equation*}
    Its solutions are
    \begin{equation*} \lambda = 1, 1 \; (\text{repeated}) \end{equation*}
    We thus have one solution as \(y_1= x.\) By the reduction of the order formula (3.2.19), a second linearly independent solution \(y_2\) is given by
    \begin{equation*} \begin{split} y_2&= y_1 \int \frac{e^{-\int a_1(x)\, dx}}{y_1^2}\, dx\quad \mbox{with } a_1(x) = -\frac{1}{x}\\ &= x \int \frac{e^{-\int \left(-\frac{1}{x}\right)\, dx}}{x^2}\, dx\\ &=x \int \frac{1}{x}\, dx\\ &= x \ln x. \end{split} \end{equation*}
    Hence the general solution of the Cauchy-Euler equation is
    \begin{equation*} y= c_1 x+c_2 x\ln x,\quad x>0, \end{equation*}
    where \(c_1\) and \(c_2\) are arbitrary constants.

      🔗
    2. Differentiate both sides of the equation in Item 3.2.3.6.a with respect to \(\lambda\) to obtain
      \begin{equation*} a\frac{\partial^2}{\partial x^2} (x e^{\lambda x}) + b\frac{\partial}{\partial x} (x e^{\lambda x}) + c (x e^{\lambda x}) = \big(p'(\lambda) + x\, p(\lambda)\big) e^{\lambda x}. \end{equation*}
      🔗
      🔗
    3. Using \(p(\lambda_0) = p'(\lambda_0) = 0\text{,}\) conclude that \(y_2 = x e^{\lambda_0 x}\) is also a solution.
      🔗
      🔗
    4. Show that \(y_1 =e^{\lambda_0 x}\) and \(y_2=x e^{\lambda_0 x}\) are linearly independent on \((-\infty, \infty)\text{.}\)
      🔗
      🔗

    7.

    Let \(y_1(x)\) be a known solution of
    \begin{equation} y'' + a_1(x)y' + a_0(x)y = 0,\tag{3.2.22} \end{equation}
    where \(a_0(x)\) and \(a_1(x)\) are continuous functions on an interval \(I,\) satisfying \(y_1(x)>0\) for all \(x\) in \(I\text{.}\) Use the Wronskian to derive Liouville’s formula (3.2.19) as follows:
    1. Suppose that \(y_2\) is a solution of (3.2.22) such that \(\{y_1, y_2\}\) is linearly independent on \(I.\) Show that
      \begin{equation*} \left(\frac{y_2}{y_1}\right)' = \frac{W(x)}{y_1^2} \quad \mbox{on } I, \end{equation*}
      where \(W(x)\) is the Wronskian of \(y_1\) and \(y_2\) given by \(W(x) =y_1(x)y_2'(x) - y_1'(x)y_2(x).\)
      🔗
      🔗
    2. Using Item 3.2.3.7.a, show that
      \begin{equation*} y_2 = y_1(x) \int\frac{W(x)}{[y_1(x)]^2}\, dx. \end{equation*}
      🔗
      🔗
    3. Obtain Liouville’s formula
      \begin{equation*} y_2 = y_1(x) \int\frac{e^{-\int a_1(x)\,dx}}{[y_1(x)]^2}\, dx \end{equation*}
      from part Item 3.2.3.7.b.
      🔗
      🔗
    The same steps also apply when the known solution \(y_1\) satisfies \(y_1(x) <0\) for all \(x\) in \(I\text{.}\) For the general case, see (ii) of Item 2.
