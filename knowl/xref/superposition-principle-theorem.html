Theorem 3.1.5. (Superposition Principle).

Suppose that \(y_1, \dots, y_k\) are solutions of the homogeneous linear equation
\begin{equation*} y^{(n)} + a_{n-1}(x)y^{(n-1)} + \cdots + a_1(x)y' + a_0(x)y = 0, \end{equation*}
where \(a_0(x), \dots, a_{n-1}(x)\) are continuous on some interval \(I .\) Then, for any constants \(c_1, \dots, c_k\) (possibly complex), the function
\begin{equation*} y(x) = c_1y_1(x) + \cdots+c_k y_k(x) = \sum_{j=1}^k c_j y_j(x) \end{equation*}
is a solution on \(I.\) The expression \(\sum_{j=1}^k c_j y_j(x)\) is called a linear combination (or superposition) of \(y_1, \dots, y_k,\) with weights \(c_1, \dots, c_k.\)
