Example 3.3.18.

Find a particular solution of
\begin{equation*} y''+(x+1)y'+xy=e^{-x^2/2}. \end{equation*}
🔗
Solution.
The equation in the operator form is
\begin{equation} p(D)y =e^{-x^2/2},\tag{3.3.27} \end{equation}
where
\begin{equation*} p(D)= D^2+(x+1)D+x. \end{equation*}
By inspection, \(p(D)\) may be factored as \((D+x)(D+1)\) or \((D+1)(D+x),\) and so which one is it? Direct computation shows
\begin{equation*} \begin{split} (D+x)(D+1)y &= (D+x)(Dy+y)\\ &= D(Dy+y)+x(Dy+y)\\ &= D^2y+Dy+xDy+xy\\ &= (D^2+(x+1)D+x)y\\ &=p(D)y, \end{split} \end{equation*}
and
\begin{equation*} \begin{split} (D+1)(D+x)y &= (D+1)(Dy+xy)\\ &= D(Dy+xy)+(Dy+xy)\\ &= D^2y+xDy+y+Dy+xy\\ &= (D^2+(x+1)D+(x+1))y\\ &\ne p(D)y. \end{split} \end{equation*}
Thus, we only have
\begin{equation*} p(D)y= (D+x)(D+1)y. \end{equation*}
Then (3.3.27) turns into
\begin{equation*} (D+x)(D+1)y= e^{-\frac{1}{2}x^2}. \end{equation*}
Put \((D+1)y =z.\) Then we have
\begin{equation*} (D+x)z=e^{-x^2/2}, \end{equation*}
which is a first order linear equation in \(z.\) In view of (3.3.21), a particular solution of this equation is given by
\begin{equation*} z_p=e^{-x^2/2}\int e^{-x^2/2} e^{x^2/2}\,dx= (x-1) e^{-x^2/2}. \end{equation*}
The constant of integration is chosen so as to simplify the integral below. Using this \(z_p\) for \(z\) in \((D+1)y =z,\) we obtain
\begin{equation*} (D+1)y = (x-1) e^{-x^2/2}. \end{equation*}
By (3.3.21), a particular solution of this equation is given by
\begin{equation*} \begin{split} y_p &= e^{-x} \int (x-1)e^x e^{-x^2/2}\, dx\\ & = e^{-x+\frac{1}{2}} \int (x-1)e^{-\frac{1}{2}(x-1)^2}\, dx\\ &=e^{-x+\frac{1}{2}} \int e^{-t}\,dt\quad \left(\mbox{using } t= \frac{1}{2}(x-1)^2 \mbox{ so that }(x-1)dx =dt\right)\\ &=-e^{-(x-\frac{1}{2})} e^{-\frac{1}{2}(x-1)^2}\\ &=-e^{-\frac{1}{2}x^2}. \end{split} \end{equation*}
