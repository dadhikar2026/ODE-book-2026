Example 3.3.8.

Find a particular solution of
\begin{equation} y'' -6y'+ 9y = \sin x\tag{3.3.12} \end{equation}
and then write down the general solution.
🔗
Solution.
First, we use the same trial solution as in Example 3.3.6 and see what happens. Let
\begin{equation*} y_p = A\sin x \end{equation*}
be a particular solution, where \(A\) is a constant to be determined. Differentiating yields
\begin{equation*} y_p' = A\cos x, \quad y_p''= -A\sin x. \end{equation*}
Substituting these into (3.3.12) would give
\begin{equation*} -A\sin x-6A\cos x+ 9A\sin x = \sin x, \end{equation*}
i.e.,
\begin{equation*} (8A-1)\sin x -6A\cos x = 0, \end{equation*}
which is not an identity in \(x\) for any value of \(A\) because \(\sin x\) and \(\cos x\) are linearly independent. Thus, no value of \(A\) makes \(y_p = A\sin x\) a valid particular solution. This occurs because \(\cos x\) is involved in the process. Therefore, we modify our trial function for \(y_p\) and take
\begin{equation*} y_p = A\sin x + B\cos x \end{equation*}
where \(A\) and \(B\) are constants to be determined. Differentiating twice gives
\begin{equation*} y_p' = A\cos x-B\sin x, \quad y_p''= -A\sin x - B\cos x. \end{equation*}
Substituting these into (3.3.12) gives
\begin{equation*} (-A\sin x - B\cos x)-6(A\cos x-B\sin x)+9(A\sin x + B\cos x) = \sin x, \end{equation*}
i.e.,
\begin{equation*} (8A+6B) \sin x +(8B-6A)\cos x =\sin x, \end{equation*}
an identity in \(x.\) Since \(\sin x\) and \(\cos x\) are linearly independent, we obtain
\begin{equation*} \begin{cases} 8A+6B=1,\\ 8B-6A=0. \end{cases} \end{equation*}
Solving these equations for \(A\) and \(B\) yields \(A = \frac{2}{25}\) and \(B= \frac{3}{50}.\) Thus,
\begin{equation*} y_p =\frac{2}{25}\sin x+\frac{3}{50}\cos x \end{equation*}
is a particular solution.
🔗
To determine the complementary function of the general solution of (3.3.12), the auxiliary equation for the associated homogeneous equation
\begin{equation*} y'' -6y'+ 9y =0 \end{equation*}
is
\begin{equation*} \lambda^2 -6\lambda +9 =0, \end{equation*}
which has a repeated root \(\lambda = 3.\) Then the complementary function \(y_h\) is given by
\begin{equation*} y_h = c_1 e^{3x} + c_2 xe^{3x}, \end{equation*}
where \(c_1\) and \(c_2\) are arbitrary constants. Hence the general solution of (3.3.12) is
\begin{equation*} y = y_h(x)+y_p(x) = c_1 e^{3x} + c_2 xe^{3x}+\frac{2}{25}\sin x+\frac{3}{50}\cos x, \end{equation*}
where \(c_1\) and \(c_2\) are arbitrary constants.
