Example 3.3.4.

Solve the initial value problem
\begin{equation*} y'' + 8 y' + 16 y = 4 x,\quad y(0) = 0,\; y'(0) = 0 \end{equation*}
by variation of parameters.
🔗
Solution.
The auxiliary equation for the associated homogeneous equation is \(\lambda^2 + 8\lambda + 16 = 0\) which gives \(\lambda = -4, -4,\) and so the functions
\begin{equation*} y_1(x) = e^{-4x}, \quad y_2(x) = x e^{-4x} \end{equation*}
form a fundamental set solutions to \(y'' + 8 y' + 16 y=0.\) The Wronskian of \(y_1\) and \(y_2\) is
\begin{equation*} W(x) = \begin{vmatrix} e^{-4x} & x e^{-4x} \\ -4 e^{-4x} & (1 - 4x) e^{-4x} \end{vmatrix} = e^{-8x} (1 - 4x + 4x) = e^{-8x}. \end{equation*}
We note that the given nonhomogeneous equation is in the standard form with \(f(x) = 4x\) and therefore a particular solution by the method of variation of parameters is given by
\begin{equation*} y_p = -y_1 \int \frac{y_2 f}{W} dx + y_2 \int \frac{y_1 f}{W} dx. \end{equation*}
Then the integration by parts with the Kronecker method (see Figure 3.3.5) gives
🔗
\begin{equation*} \int \frac{y_2 f}{W} dx= \int \frac{4x^2 e^{-4x}}{e^{-8x}}\, dx =x^2 e^{4x} - \frac{1}{2} x e^{4x} + \frac{1}{8} e^{4x}. \end{equation*}
In a similar manner, we also evaluate
\begin{equation*} \int \frac{y_1 f}{W} dx=\int \frac{4x e^{-4x}}{e^{-8x}}\, dx= \int 4x e^{4x} dx = x e^{4x} - \frac{1}{4} e^{4x}. \end{equation*}
Then \(y_p\) becomes
\begin{equation*} \begin{aligned} y_p &= -e^{-4x} \left( x^2 e^{4x} - \frac{1}{2} x e^{4x} + \frac{1}{8} e^{4x} \right) + x e^{-4x}\left( x e^{4x} - \frac{1}{4} e^{4x} \right)\\ &=- x^2 + \frac{x}{2} - \frac{1}{8} + x^2 - \frac{x}{4}\\ &=\frac{x}{4} - \frac{1}{8}. \end{aligned} \end{equation*}
The general solution of the nonhomogeneous differential equation is
\begin{equation*} y(x) = c_1 e^{-4x} + c_2 x e^{-4x} + \frac{x}{4} - \frac{1}{8}, \end{equation*}
where \(c_1\) and \(c_2\) are arbitrary constants. Since \(y(0) = 0,\) we get \(c_1 = \frac{1}{8}.\) We find
\begin{equation*} y'(x) = -4 c_1 e^{-4x} + c_2 e^{-4x} - 4 c_2 x e^{-4x} + \frac{1}{4}. \end{equation*}
Since \(y'(0) =0,\) we have\(-4 c_1 + c_2 + \frac{1}{4} = 0.\) Using \(c_1= \frac{1}{8}\text{,}\) we get \(-\frac{1}{2} + c_2 + \frac{1}{4} = 0\text{,}\) so that \(c_2 = \frac{1}{4}.\) Hence the solution to the given initial value problem is
\begin{equation*} y(x) = \frac{1}{8} e^{-4x} + \frac{1}{4} x e^{-4x} + \frac{x}{4} - \frac{1}{8} \end{equation*}
on \((-\infty, \infty).\)
