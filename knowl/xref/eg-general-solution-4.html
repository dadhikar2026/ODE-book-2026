Example 3.2.10.

Find the general solution of
\begin{equation*} y''+8y'+16y = 0. \end{equation*}
🔗
Solution.
Assume a solution of the form \(y = e^{\lambda x}\text{.}\) Then \(y' = \lambda e^{\lambda x}\) and \(y'' = \lambda^2 e^{\lambda x}\text{.}\) Substituting into the differential equation gives
\begin{equation*} \lambda^2 e^{\lambda x} + 8\lambda e^{\lambda x} + 16 e^{\lambda x} = 0, \end{equation*}
which gives
\begin{equation*} (\lambda^2 + 8\lambda + 16) e^{\lambda x} = 0. \end{equation*}
Since \(e^{\lambda x} \neq 0\text{,}\) the auxiliary equation that we obtain is
\begin{equation*} \lambda^2 + 8\lambda + 16 = 0, \end{equation*}
whose repeated real root is
\begin{equation*} \lambda = -4. \end{equation*}
By (3.2.20), the general solution is
\begin{equation*} y(x) = (c_1 + c_2 x) e^{-4x}, \end{equation*}
where \(c_1\) and \(c_2\) are arbitrary constants.
