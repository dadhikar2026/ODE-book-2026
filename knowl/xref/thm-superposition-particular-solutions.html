Theorem 3.3.11.

Let \(a_0(x), a_1(x), \dots, a_{n-1}(x)\) and \(f_i(x), \quad i = 1, \dots, k,\) be continuous on an interval \(I.\) Suppose \(y_{p_i},\; i = 1, \dots, k,\) is a solution of
\begin{equation} y^{(n)}+ a_{n-1}(x)y^{(n-1)}+ \cdots+ + a_1(x) y' + a_0(x) y = f_i(x),\tag{3.3.15} \end{equation}
for each \(i = 1, \dots, k.\) Then the function
\begin{equation*} y_p= \sum_{j=1}^ky_{p_j}(x) \end{equation*}
is a solution of
\begin{equation} y^{(n)}+ a_{n-1}(x)y^{(n-1)}+ \cdots + a_1(x) y' + a_0(x) y = \sum_{j=1}^kf_j(x)\tag{3.3.16} \end{equation}
on \(I\)
