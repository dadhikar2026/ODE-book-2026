Definition 3.1.8.

The functions \(y_1, \dots, y_k\) defined on an interval \(I\text{,}\) where \(k\) is a positive integer, are said to be linearly independent on \(I\) if the relation
\begin{equation*} c_1 y_1(x) + \cdots + c_k y_k(x) = 0 \end{equation*}
holds for all \(x\) in \(I\) only when the constants \(c_1, \dots, c_k\) satisfy \(c_1 = \cdots = c_k = 0\text{.}\) If this condition does not hold, that is, if there exist constants \(c_1, \dots, c_k\text{,}\) not all zero, satisfying the above relation for all \(x\) in \(I\text{,}\) then the functions \(y_1, \dots, y_k\) are said to be linearly dependent on \(I\).
