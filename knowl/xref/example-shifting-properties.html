Example 3.3.17.

Find a particular solution of
\begin{equation*} y'' - 5y' + 6y = 4x e^x \end{equation*}
by using the exponential shifting properties.
🔗
Solution.
We can write
\begin{equation*} 4x e^x = e^x (4x). \end{equation*}
By the exponential shifting property of the inverse operator, we have
\begin{equation*} y_p = \frac{1}{D^2 - 5D + 6} (4x e^x) = e^x \frac{1}{(D+1)^2 - 5(D+1) + 6} (4x). \end{equation*}
Since
\begin{equation*} (D+1)^2 - 5(D+1) + 6 = D^2 - 3D + 2, \end{equation*}
we have
\begin{equation*} y_p = e^x \frac{1}{D^2 - 3D + 2} (4x). \end{equation*}
By using partial fraction decomposition and then applying (3.3.22) and integration by parts, we get
\begin{equation*} \begin{split} y_p&= e^x \left(\frac{A_1}{D-1}+\frac{A_2}{D-2}\right)4x\\ &= e^x\left(-\frac{4}{D-1}+\frac{4}{D-2}\right)x\\ &=e^x\left(-4e^{x}\int xe^{-x}\, dx +4e^{2x}\int xe^{-2x}\, dx\right)\\ &=(2x+3)e^x.\end{split} \end{equation*}
