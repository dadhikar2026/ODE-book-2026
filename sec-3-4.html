    Section 3.4 Higher Order Linear Differential Equations

    Objectives

    • Learn solution techiques for higher order linear differential equations
      🔗
      🔗
    🔗
    In this section we discuss methods for solving the \(n^\text{th}\) order linear differential equations of the form
    \begin{equation} y^{(n)} + a_{n-1}(x)y^{(n-1)} + \cdots + a_1(x)y' + a_0(x)y = f(x),\tag{3.4.1} \end{equation}
    where the coefficient functions \(a_0(x), \dots, a_{n-1}(x)\) and the forcing term \(f(x)\) are continuous on some interval \(I .\)
    🔗

    Subsection 3.4.1 Homogeneous Equations

    To find the general solution of (3.4.1), we first find the general solution \(y_h\) of the associated homogeneous equation
    \begin{equation} y^{(n)} + a_{n-1}(x)y^{(n-1)} + \cdots + a_1(x)y' + a_0(x)y = 0\tag{3.4.2} \end{equation}
    and a particular solution \(y_p\) of (3.4.1). Then the general solution of the nonhomogeneous equation (3.4.1) is given by
    \begin{equation} \boxed{ \begin{split}y &= y_h(x)+y_p(x)\\ & = c_1 y_1(x) + c_2 y_2(x) + \cdots + c_n y_n(x) + y_p(x)\end{split}}\tag{3.4.3} \end{equation}
    on \(I,\) where \(\{y_1, y_2, \dots, y_n\}\) is a fundamental set of solutions of (3.4.2) and \(c_1, \dots, c_n\) are arbitrary constants. The function \(y_h(x),\) also denoted by \(y_c(x),\) is called the complimentary function of the general solution of (3.4.2). The existence of a fundamental set of solutions for (3.4.2) follows by arguments analogous to those used in Theorem 3.2.3. In fact, let \(y_i\) be the unique solution of the initial value problem consisting of the equation (3.4.2) subject to the initial conditions
    \begin{equation} y(x_0) =0, \dots, y^{(i-1)}(x_0) =1, \dots, y^{(n-1)}(x_0) =0, \end{equation}\tag{3.4.4} \end{equation}
    where \(i = 1, \dots, n.\) It follows that \(\{y_1, \dots, y_n\}\) is a fundamental solutions of (3.4.2) (see Exercise 3.4.3.4 in Exercises 3.4.3).
    🔗
    Let \(y_1, \dots, y_n\) be solutions of (3.4.2). In Exercise 3.4.3.5 in Exercises 3.4.3, you will derive the Abel formula for the Wronskian of \(y_1, \dots, y_n\) as
    \begin{equation*} W(y_1, \dots, y_n) (x)= W(y_1, \dots, y_n)(x_0) e^{\int_{x_0}^x a_{n-1}(t)\, dt}, \; x\in I. \end{equation*}
    🔗
    The linear independence of solutions \(y_1, y_2, \dots, y_n\) of (3.4.2) can be determined using their Wronskian (see Definition 3.1.11), as stated in the following theorem.
    🔗

    Theorem 3.4.1.

    Let \(y_1, y_2, \dots, y_n\) be solutions of (3.4.2) on \(I.\) Then they are linearly independent on \(I\) if and only if
    \begin{equation*} W(y_1, \dots, y_n)(x)\ne 0 \mbox{ for all } x \mbox{ in } I. \end{equation*}
    🔗
    🔗

    Proof.

    The proof is similar to the proof of Theorem 3.2.1 and is omiited.
    🔗
    🔗
    Methods for constructing a fundamental set of solutions for (3.4.2) vary. When we have means to find a single nonzero solution, the reduction of order (see Paragraphs  ) can be employed to obtain additional linearly independent solutions. This method applies not only to second order homogeneous linear equations but also extends to higher order equations, as discussed in the next theorem.
    🔗

    Theorem 3.4.2.

    Let \(y_1\) be a solution of (3.4.2) such that \(y_1(x)\ne 0\) on \(I,\) and let \(y = u y_1\) be a solution of (3.4.2) for a nonconstant function \(u\) on \(I.\) If
    \begin{equation*} \{v_1, \dots, v_{n-1}\} \end{equation*}
    is a fundamental set of solutions for the equation of order \(n-1\) in \(v=u'\) obtained by substituting \(y= u y_1\) into (3.4.2) and if \(u'_k= v_k\) for \(k = 1, \dots, n-1,\) then
    \begin{equation*} \{y_1, u_1y_1, u_2y_1, \dots, u_{n-1}y_1\} \end{equation*}
    forms a fundamental set of solutions for (3.4.2) on \(I.\)
    🔗
    🔗

    Proof.

    We present a proof for the case \(n=3.\) The general case proceeds in a similar way, and a careful reader is encouraged to verify it, noting that it involves the use of binomial coefficients that are produced for higher order derivatives of the product function \(y= u y_1.\)
    🔗
    For \(y= uy_1\) to be a solution of (3.4.2), we must have
    \begin{equation*} \begin{split} &u'''y_1+ 3u''y'_1+ 3u'y''_1 + uy'''_1\\ &+a_2 u''y_1+2a_2 u'y'_1+a_2uy''_1\\ &+a_1u'y_1+a_1uy'_1\\ &+a_0uy_1 =0, \end{split} \end{equation*}
    which gives
    \begin{equation*} \begin{split} &u'''y_1+ (3y'_1+a_2y_1) u''+ (3y''_1+2a_2y'_1+a_1y_1)u'\\ &+ (y'''_1+a_2y''_1+a_1y'_1+a_0y_1)u=0 \end{split} \end{equation*}
    Since \(y_1\) is a solution of (3.4.2), we have \(y'''_1+a_2y''_1+a_1y'_1+a_0y_1=0.\) Consequently, we obtain
    \begin{equation*} u'''y_1+ (3y'_1+a_2y_1) u''+ (3y''_1+2a_2y'_1+a_1y_1)u'=0. \end{equation*}
    Letting \(v= u'\) gives
    \begin{equation*} v''y_1+ (3y'_1+a_2y_1) v'+ (3y''_1+2a_2y'_1+a_1y_1)v=0, \end{equation*}
    which is a second order homogeneous linear equation in \(v.\) Suppose that \(\{v_1, v_2\}\) is a fundamental set of solution of the second order equation and let \(u_k\) be functions satisfying \(u'_k = v_k,\; k = 1, 2.\) To show
    \begin{equation*} \{y_1, u_1y_1, u_2y_1 \} \end{equation*}
    is a fundamental set of solutions for (3.4.2), let \(c_0, c_1, c_2\) be real numbers such that
    \begin{equation*} c_0y_1(x) +c_1u_1(x) y_1(x) + c_2 u_2(x) y_1(x) = 0 \end{equation*}
    for all \(x\) in \(I.\) Since \(y_1(x) \ne 0\) on \(I,\) we must have
    \begin{equation*} c_0 +c_1u_1(x) + c_2 u_2(x) = 0 \end{equation*}
    for all \(x\) in \(I.\) Differentiating yields
    \begin{equation*} c_1u'_1(x) + c_2u'_2(x) = 0 \end{equation*}
    for all \(x\) in \(I,\) that is,
    \begin{equation*} c_1v_1(x) + c_2v_2(x) = 0 \end{equation*}
    for all \(x\) in \(I.\) The linear independence of \(\{v_1, v_2\}\) on \(I\) implies that \(c_1 = c_2 = 0.\) Consequently, \(c_0 = 0\) as well. Therefore,
    \begin{equation*} \{y_1,\, u_1 y_1,\, u_2 y_1\} \end{equation*}
    forms a linearly independent set of solutions of (3.4.2) for \(n = 3,\) and hence constitutes a fundamental set of solutions. ◻
    🔗
    🔗

    Example 3.4.3.

    Find a fundamental set of solutions for
    \begin{equation} y^{(4)}-4y'''+6y''-4y'+y=0,\tag{3.4.5} \end{equation}
    given that \(y= e^x\) is a solution. Also, write down the general solution.
    🔗
    Solution.
    Let \(y= ue^x\) a solution of (3.4.5), where \(u\) is a nonconstant function on \((-\infty, \infty).\) Differentiating four times and substituting into the equation gives
    \begin{equation*} \begin{split} &u^{(4)}e^x+ 4u'''e^x+ 6u''e^x + 4u'e^x+ ue^x\\ &-4u'''e^x -12u''e^x-12u'e^x -4 ue^x\\ &+6u''e^x +12u'e^x+6 ue^x\\ &-4u'e^x -4ue^x \\ &+ue^x =0. \end{split} \end{equation*}
    After canceling terms, we obtain \(u^{(4)}e^x=0,\) which yields \(u^{(4)}(x)=0\) in \((-\infty, \infty).\) Let \(v= u'.\) Then we find \(v'''(x) = 0\) on \(I.\) Integrating, we obtain
    \begin{equation*} v(x) = Ax^2+ Bx + C, \end{equation*}
    where \(A, B, C\) are arbitrary constants.
    🔗
    Take \(v_1(x) = 1, v_2(x)= 2x,\) and \(v_3(x)= 3x^2.\) Then \(\{v_1, v_2, v_3\}\) forms a fundamental set of solutions for \(v'''(x) = 0\) on \(I.\) Since \(u'=v,\) we take \(u_1(x) = x, u_2(x) = x^2,\) and \(u_3(x) =x^3,\) so that
    \begin{equation*} \{e^x, xe^x, x^2e^x, x^3 e^x\} \end{equation*}
    forms a fundamental set of solutions for (3.4.5). The general solution of the equation is
    \begin{equation*} y= c_1e^x+c_2xe^x+c_3x^2e^x+c_4x^3 e^x = e^x(c_1+c_2x+c_3x^2+c_4x^3), \end{equation*}
    where \(c_1, c_2, c_3, \text{ and } c_4\) are arbitrary constants.
    🔗
    🔗
    🔗
    🔗

    Subsection 3.4.2 Nonhomogeneous Equations

    To find a particular solution \(y_p(x)\) of (3.4.1) by using the method of variation of parameters, we take \(y_p(x)\) as
    \begin{equation} y_p(x) = u_1(x)y_1(x) + u_2(x)y_2(x) + \cdots + u_n(x)y_n(x),\tag{3.4.6} \end{equation}
    where \(u_1(x), u_2(x), \dots, u_n(x)\) are to be determined.
    🔗
    Differentiating (3.4.6) successively, and to simplify calculations, imposing the conditions
    \begin{equation*} \begin{split} u_1'y_1 + u_2'y_2 + \cdots + u_n'y_n &= 0, \\ u_1'y_1' + u_2'y_2' + \cdots + u_n'y_n' &= 0, \\ &\vdots \\ u_1'y_1^{(n-2)} + u_2'y_2^{(n-2)} + \cdots + u_n'y_n^{(n-2)} &= 0, \end{split} \end{equation*}
    we obtain
    \begin{equation*} u_1'y_1^{(n-1)} + u_2'y_2^{(n-1)} + \cdots + u_n'y_n^{(n-1)} = f(x). \end{equation*}
    These \(n\) equations can be solved for \(u_1', u_2', \dots, u_n'\) by using Cramer’s rule as follows. We first write down the system of \(n\) equations in the matrix form
    \begin{equation} \begin{bmatrix} y_1 & y_2 & \cdots & y_n \\ y_1' & y_2' & \cdots & y_n' \\ \vdots & \vdots & \ddots & \vdots \\ y_1^{(n-1)} & y_2^{(n-1)} & \cdots & y_n^{(n-1)} \end{bmatrix} \begin{bmatrix} u_1' \\ u_2' \\ \vdots \\ u_n' \end{bmatrix} = \begin{bmatrix} 0 \\ 0 \\ \vdots \\ f(x) \end{bmatrix},\tag{3.4.7} \end{equation}
    which can be solved for \(u_1', u_2', \dots, u_n'\) by using Cramer’s rule. In fact, we have
    \begin{equation*} u'_j= \frac{W_j}{W}, \end{equation*}
    where \(W\) is the Wronskian of \(y_1, \dots, y_n\) and \(W_k\) is the determinant obtained by replacing the \(j-\)th column of \(W\) by the column on the right-hand side of (3.4.7). We find
    \begin{equation*} u_j(x) =\int \frac{W_j(x)}{W(x)}\, dx \end{equation*}
    for each \(j = 1, \dots, n\) and hence \(y_p(x)\) is then fully determined by (3.4.6).
    🔗
    As discussed in Remark 3.3.3, the method of variation of parameters is quite general and works for any linear nonhomogeneous differential equation (even with variable coefficients), so long as the required integrals for \(u_1, u_2, \dots, u_n\) make sense.
    🔗

    Example 3.4.4.

    Find the general solution of
    \begin{equation*} y''' + y' = \tan x \end{equation*}
    by using the variation of parameters.
    🔗
    Solution.
    The homogeneous equation associated to the nonhomogeneous equation \(y''' + y' = \tan x\) is
    \begin{equation*} y''' + y' = 0 \end{equation*}
    with the auxiliary equation
    \begin{equation*} r^3 + r = 0 \quad \text{ so that } \quad r(r^2 + 1) = 0. \end{equation*}
    Then we have
    \begin{equation*} r = 0, \quad r = i, \quad r = -i, \end{equation*}
    and therefore the functions
    \begin{equation*} y_1(x) = 1, \quad y_2(x) = \sin x, \quad y_3(x) = \cos x \end{equation*}
    form a fundamental set of solutions for the homogeneous equation. The general solution to \(y'''+y= \tan x\) is then given by
    \begin{equation*} y(x) = c_1 + c_2 \sin x + c_3 \cos x + y_p(x), \end{equation*}
    where \(y_p(x)\) is a particular solution of the nonhomogeneous equation.
    🔗
    By the method of variation of parameters, we assume that \(y_p(x)\) is of the form
    \begin{equation*} y_p(x) = u_1(x) + u_2(x) \sin x + u_3(x) \cos x, \end{equation*}
    where the functions \(u_1, u_2, u_3\) are to be determined so that \(u_1', u_2', u_3'\) satisfy
    🔗
    \begin{equation*} \begin{cases} u_1' + u_2' \sin x + u_3' \cos x = 0, \\[6pt] u_2' \cos x - u_3' \sin x = 0, \\[6pt] u_2' \sin x - u_3' \cos x= \tan x. \end{cases} \end{equation*}
    The system in the matrix form is
    \begin{equation*} \begin{bmatrix} 1 & \sin x & \cos x \\ 0 & \cos x & -\sin x \\ 0 & -\sin x & -\cos x \end{bmatrix} \begin{bmatrix} u_1' \\ u_2' \\ u_3' \end{bmatrix} = \begin{bmatrix} 0 \\ 0 \\ \tan x \end{bmatrix}. \end{equation*}
    The Wronskian \(W(x)\) of \(1, \sin x, \cos x\) is
    \begin{equation*} W(x) = \begin{vmatrix} 1 & \sin x & \cos x\\[6pt] 0 & \cos x & -\sin x \\[6pt] 0 & -\sin x & -\cos x \end{vmatrix}. \end{equation*}
    Using the co-factor expansion along the first column, we have
    \begin{equation*} W(x) = \begin{vmatrix} \cos x & -\sin x \\[6pt] -\sin x& -\cos x \end{vmatrix}\\ = -\cos^2 x - \sin^2 x\\ = -1. \end{equation*}
    We compute
    \begin{equation*} W_1(x) = \begin{vmatrix} 0 & \sin x & \cos x\\[6pt] 0 & \cos x& -\sin x \\[6pt] \tan x & -\sin x & -\cos x \end{vmatrix} \end{equation*}
    by expanding along the first column and obtain
    \begin{equation*} W_1(x) = \tan x \begin{vmatrix} \sin x & \cos x \\[6pt] \cos x & -\sin x \end{vmatrix} = \tan x( -\sin^2 x- \cos^2 x ) = -\tan x. \end{equation*}
    Then we have
    \begin{equation*} u_1' = \frac{W_1}{W} = \frac{ -\tan x }{ -1 } = \tan x, \end{equation*}
    and consequently, we can take
    \begin{equation*} u_1(x) = -\ln\lvert{\cos x}\rvert. \end{equation*}
    We next compute
    \begin{equation*} W_2(x) = \begin{vmatrix} 1 & 0 & \cos x \\[6pt] 0 & 0 & -\sin x\\[6pt] 0 & \tan x& -\cos x \end{vmatrix} \end{equation*}
    by expanding along the first column and obtain
    \begin{equation*} W_2(x) = \begin{vmatrix} 0 & -\sin x \\[6pt] \tan x & -\cos x \end{vmatrix} = 0 - (-\sin x) \tan x = \sin x \tan x. \end{equation*}
    Then we have
    \begin{equation*} u_2' = \frac{W_2}{W} = \frac{ \sin x \tan x }{ -1 } = -\sin x \tan x =-\frac{\sin^2x}{\cos x}= \cos x- \sec x, \end{equation*}
    and so
    \begin{equation*} u_2(x) = \int \big(\cos x- \sec x\big)\, dx. \end{equation*}
    We can take
    \begin{equation*} u_2(x)= \sin x- \ln\lvert{\sec x+\tan x}\rvert. \end{equation*}
    We also compute
    \begin{equation*} W_3(x) = \begin{vmatrix} 1 & \sin x & 0 \\[6pt] 0 & \cos x & 0 \\[6pt] 0 & -\sin x & \tan x \end{vmatrix} \end{equation*}
    by expanding along the first column and obtain
    \begin{equation*} W_3(x) = \begin{vmatrix} \cos x & 0 \\[6pt] -\sin x & \tan x \end{vmatrix} = \cos x \tan x = \sin x, \end{equation*}
    and so
    \begin{equation*} u_3' = \frac{W_3}{W} = \frac{ \cos x \tan x }{ -1 } = - \sin x. \end{equation*}
    We can take
    \begin{equation*} u_3(x) = \cos x. \end{equation*}
    Consequently, \(y_p\) is given by
    \begin{equation*} \begin{aligned} y_p(x) &= u_1(x) + u_2(x) \sin x + u_3(x) \cos x\\ &= \ln\lvert{\sec x}\rvert+ \sin x\big(\sin x-\ln\lvert{\sec x+\tan x}\rvert\big)+\cos^2(x)\\ &=1+\ln\lvert{\sec x}\rvert-\sin x\ln\lvert{\sec x+\tan x}\rvert. \end{aligned} \end{equation*}
    Hence the general solution to \(y'''+y' = \tan x\) is
    \begin{equation*} y(x) = c_1 + c_2 \sin x + c_3 \cos x +\ln\lvert{\sec x}\rvert-\sin x\ln\lvert{\sec x+\tan x}\rvert, \end{equation*}
    where \(c_1\text{,}\) \(c_2\) and \(c_3\) are arbitrary constants.
