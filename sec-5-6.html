    Section 5.6 Systems of Linear Differential Equations

    This chapter focuses on the applications of the Laplace transform to solve systems of linear ordinary differential equations. Using the Laplace transform, we convert a system of differential equations into a system of algebraic equations involving Laplace transforms of the unknown functions (dependent variables), solve this algebraic system for the Laplace transforms , and then apply the inverse Laplace transform to recover the functions from their Laplace transforms. These functions then become solutions of the differential equations in the system.
    🔗
    To fix the ideas, we consider below a first-order linear system of ordinary differential equations in the three variables \(x(t), y(t), z(t)\text{:}\)
    \begin{equation} \begin{cases} \ds \frac{dx}{dt}\amp= a_{11}x + a_{12}y + a_{13}z + f_1(t),\\ \ds \frac{dy}{dt} \amp= a_{21}x + a_{22}y + a_{23}z + f_2(t),\\ \ds \frac{dz}{dt} \amp= a_{31}x + a_{32}y + a_{33}z + f_3(t), \end{cases}\tag{5.6.1} \end{equation}
    where
    • \(x(t), y(t), z(t)\) are unknown functions of time \(t\text{,}\)
      🔗
      🔗
    • the coefficients \(a_{ij}\) are fixed real numbers, and
      🔗
      🔗
    • \(f_i(t)\) are given functions of \(t\text{,}\) possibly zero.
      🔗
      🔗
    🔗
    The system (5.6.1) is said to be decoupled if \(a_{ij}= 0\) for all \(i, j\) with \(i\ne j\) and coupled if for each \(i = 1, 2, 3\) there is \(j = 1, 2,3\) with \(i\ne j\) and \(a_{ij}\ne 0.\) The linear system (5.6.1) is said to be partially decoupled if it has a structure that allows solving one or more of the equations in isolation, and then using those solutions to simplify or solve the remaining equations.
    🔗

    Subsection 5.6.1 The Laplace Transform Method

    To solve the linear system (5.6.1) with the initial conditions
    \begin{equation*} x(0) = x_0, \quad y(0) = y_0, \quad z(0) = z_0, \end{equation*}
    we apply the Laplace transform \(\mathcal{L}\) to each equation. The system is then transformed into
    🔗
    \begin{equation*} \begin{cases} sX(s) - x_0 = a_{11}X(s) + a_{12}Y(s) + a_{13}Z(s) + F_1(s), \\ sY(s) - y_0 = a_{21}X(s) + a_{22}Y(s) + a_{23}Z(s) + F_2(s), \\ sZ(s) - z_0 = a_{31}X(s) + a_{32}Y(s) + a_{33}Z(s) + F_3(s), \end{cases} \end{equation*}
    where \(X(s), Y(s), Z(s)\) are the Laplace transforms of \(x(t), y(t), z(t)\text{,}\) and \(F_i(s) = \mathcal{L}\{f_i(t)\}, i =1, 2, 3\text{.}\)
    🔗
    We solve the above system for \(X(s), Y(s), Z(s)\) using algebraic techniques, such as Cramer’s rule, Gaussian elimination, etc. and then apply the inverse Laplace transform to obtain solutions \(x(t), y(t), z(t)\text{.}\)
    🔗
    We proceed to discuss four main examples; namely, radioactive decay series, mixing solutions, coupled spring-mass systems, and electrical circuits.
    🔗
    🔗

    Subsection 5.6.2 Radioactive Decay Series

    Consider a three-stage radioactive decay series of a radioactive element represented in a schematic diagram below:
    \begin{equation*} X \longrightarrow Y \longrightarrow Z \end{equation*}
    where
    🔗
    • \(X\) is a parent radioactive isotope,
      🔗
      🔗
    • \(Y\) is a radioactive daughter element, and
      🔗
      🔗
    • \(Z\) is a stable (non-decaying) end element.
      🔗
      🔗
    🔗
    Let
    \begin{equation*} \begin{split} x(t) &=\ \text{the amount of isotope } X \text{ at time } t, \\ y(t) &=\ \text{the amount of isotope } Y \text{ at time } t, \\ z(t) &=\ \text{the amount of isotope } Z \text{ at time } t, \\ a &=\ \text{the decay constant of } X, \text{ and} \\ b &=\ \text{the decay constant of } Y. \end{split} \end{equation*}
    We assume that the disintegration rate of \(X\) at time \(t\) is proportional to its amount at that time, and the same holds true for \(Y\text{.}\) Then the system of differential equations modeling the decay series is
    \begin{equation} \begin{cases} \ds\frac{dx}{dt} &= -a x\\[1ex] \ds\frac{dy}{dt} &= ax - by\\[1ex] \ds\frac{dz}{dt} &= by \end{cases}\tag{5.6.2} \end{equation}
    subject to the initial conditions:
    \begin{equation*} x(0) = x_0, \quad y(0) = 0, \quad z(0) = 0. \end{equation*}
    🔗
    The linear system (5.6.2) describes how the parent isotope \(X\) decays into \(Y\text{,}\) which in turn decays into a stable isotope \(Z\text{.}\) We can adopt this scheme to model real-world decay chains such as
    \begin{equation*} \text{Uranium-238} \rightarrow \text{Thorium-234}\rightarrow\cdots \rightarrow \text{Lead-206}. \end{equation*}
    The linear system (5.6.2) is an example of a partially decoupled system. We observe that the solutions of the differential equation in the system (5.6.2) can be obtained in a sequence of steps: first solve the first equation for \(x(t)\text{,}\) substitute this \(x(t)\) into the second equation to find \(y(t)\) using the integrating factor method, and then substitute \(y(t)\) into the third equation to solve for \(z(t)\text{.}\) Details are omitted, but the solutions can be found to be
    \begin{equation*} \begin{cases} x(t) &= x_0 e^{-a t}, \\[6pt] y(t) &= \frac{a x_0}{b - a} \left( e^{-a t} - e^{-b t} \right), \\[6pt] z(t) &= x_0 \left[ 1 + \frac{a e^{-b t} - b e^{-a t}}{b - a} \right]. \end{cases} \end{equation*}
    Let us obtain these solutions by using the Laplace transform method.
    🔗
    Step 1: Laplace Transforms. Since
    \begin{equation*} \mathcal{L}\left\{\frac{dx}{dt}\right\} =-a \mathcal{L}\{x(t)\}, \end{equation*}
    we have
    \begin{equation*} s X(s) - x_0 = -a X(s), \end{equation*}
    and therefore
    \begin{equation*} X(s) =\frac{x_0}{s + a}. \end{equation*}
    Similarly,
    🔗
    \begin{equation*} \mathcal{L}\left\{\frac{dy}{dt}\right\} = a X(s) - b Y(s), \end{equation*}
    which yields
    \begin{equation*} s Y(s) = a X(s) - b Y(s), \end{equation*}
    and therefore
    \begin{equation*} Y(s) = \frac{a X(s)}{s + b} = \frac{a x_0}{(s + a)(s + b)}. \end{equation*}
    Also,
    \begin{equation*} \mathcal{L}\left\{\frac{dz}{dt}\right\} = b Y(s) \end{equation*}
    gives
    \begin{equation*} Z(s) = \frac{b}{s} Y(s) = \frac{a b x_0}{s(s + a)(s + b)} \end{equation*}
    🔗
    Step 2: Inverse Laplace Transforms.We have
    \begin{equation*} x(t) = {\mathcal L}^{-1}\left\{\frac{x_0}{s + a}\right\} = x_0 e^{-a t}. \end{equation*}
    Since
    \begin{equation*} Y(s) = \frac{a x_0}{(s + a)(s + b)} = \frac{a x_0}{b - a} \left( \frac{1}{s + a} - \frac{1}{s + b} \right), \end{equation*}
    we have
    \begin{equation*} y(t) = \frac{a x_0}{b - a} \left( e^{-a t} - e^{-b t} \right). \end{equation*}
    Finally, since
    \begin{equation*} Z(s) = \frac{a b x_0}{s(s + a)(s + b)}, \end{equation*}
    we have
    \begin{equation*} \begin{split} z(t) \amp= \int_0^t b y(\tau) \, d\tau\\ \amp= \frac{a b x_0}{b - a} \left( \frac{1 - e^{-a t}}{a} - \frac{1 - e^{-b t}}{b} \right)\\ \amp= x_0 \left[ 1 + \frac{a e^{-b t} - b e^{-a t}}{b - a} \right] \end{split} \end{equation*}
    Thus, the solutions are given by
    🔗
    \begin{equation*} \begin{cases} x(t) &= x_0 e^{-a t}, \\[6pt] y(t) &= \frac{a x_0}{b - a} \left( e^{-a t} - e^{-b t} \right), \\[6pt] z(t) &= x_0 \left[\ds 1 + \frac{a e^{-b t} - b e^{-a t}}{b - a} \right], \end{cases} \end{equation*}
    as expected.
    🔗
    🔗

    🔗
    \begin{equation*} \displaystyle\begin{cases} \displaystyle x'(t) = r_{\text{in}}\; c+\frac{k_2y(t)}{V_2+t(k_1-k_2-r_{\text{out}})}-\frac{k_1x(t)}{V_1 +t(r_{\text{in}}+k_2-k_1)},\\[2ex] \displaystyle y'(t) = \frac{k_1x(t)}{V_1 +t(r_{\text{in}}+k_2-k_1)}-\frac{(k_2+r_{\text{out}})y(t)}{V_2+t(k_1-k_2-r_{\text{out}})}. \end{cases} \end{equation*}

    Subsection 5.6.4 Coupled Spring-Mass Systems

    Suppose that two masses \(m_1\) and \(m_2\) are connected to two springs \(A\) and \(B\) hanging vertically from a rigid support to form a coupled spring-mass system as shown in the figure below.
    🔗
    Let \(x_1(t)\) and \(x_2(t)\) denote the vertical displacements of the masses from their equilibrium positions. We take the positive direction of the displacements to be vertically downward. When the masses are in motion, the spring \(B\) is subject to both a stretch and a compression, so that its net stretch of the spring \(B\) is given by \(x_2-x_1\) when the stretch of the spring \(A\) is \(x_1\text{.}\) Let \(k_1\) and \(k_2\) be the constants of proportionality in Hooke’s law for the springs \(A\) and \(B\text{,}\) respectively. We assume that there is no damping or external force acting on the masses and that weights of the springs are negligible compared to weights of the masses.
    🔗
    The restoring force of the spring \(A\) acting on the mass \(m_1\) at time \(t\) is \(-k_1 x_1(t),\) and the restoring force of the spring \(B\) acting on the mass \(m_2\) at time \(t\) is \(-k_2(x_2(t)- x_1(t)).\) In addition to the restoring force of the spring \(A\text{,}\) the mass \(m_1\) also experiences the force of \(k_2(x_2(t)- x_1(t))\) at time \(t\text{.}\) Suppressing \(t\) in \(x_1(t)\) and \(x_2(t)\) and using Newton’s law, we see that the motion of the two masses in the spring-mass system is described by the coupled linear system
    \begin{equation} \begin{cases} m_1x''_1 &= -k_1 x_1 + k_2 (x_2 - x_1) = -(k_1+k_2) x_1 +k_2 x_2,\\ m_2 x''_2 &= -k_2 (x_2 - x_1)= k_2 x_1-k_2 x_2, \end{cases}\tag{5.6.3} \end{equation}
    which, together with the initial conditions \(x_1(t_0) = a, x'_1(t_0) = b\text{,}\) \(x_2(t_0) = c, x'_2(t_0) = d,\) forms an initial value problem.
    🔗
    In the next example, we solve the initial value problem (5.6.3) for \(m_1=m_2=1, k_1=3, k_2=2,\) and \(x_1(0)=0, x'_1(0)=-1, x_2(0)=0, x'_2(0)=1.\)
    🔗

    Example 5.6.3.

    Solve the second-order coupled system of linear differential equations with initial conditions
    \begin{equation} \begin{cases} x_1'' = -5 x_1 + 2 x_2 , \\ x_2'' = 2 x_1 - 2 x_2, \\ x_1(0) = 0, \quad x_1'(0) = -1, \\ x_2(0) = 0, \quad x_2'(0) = 1. \end{cases}\tag{5.6.4} \end{equation}
    🔗
    Taking the Laplace transform yields
    🔗
    \begin{equation*} \begin{cases} s^2 X_1(s) - s x_1(0) - x_1'(0) = -5 X_1(s) + 2 X_2(s), \\ s^2 X_2(s) - s x_2(0) - x_2'(0) = 2 X_1(s) - 2 X_2(s). \end{cases} \end{equation*}
    Using the given initial conditions yields
    \begin{equation*} \begin{cases} (s^2+5) X_1(s) + 1 = 2 X_2(s), \\ (s^2+2) X_2(s) - 1 = 2 X_1(s). \end{cases} \end{equation*}
    Solving for \(X_1(s)\) and \(X_2(s)\) gives
    🔗
    \begin{equation*} X_1(s) = \frac{-s^2}{(s^2 + 1)(s^2 + 6)} \quad \text{and}\quad X_2(s) = \frac{s^2 + 3}{(s^2 + 1)(s^2 + 6)}. \end{equation*}
    Partial fraction decomposition of \(X_1(s)\) and \(X_2(s)\) are
    🔗
    \begin{equation*} X_1(s) = \frac{1/5}{s^2 + 1} - \frac{6/5}{s^2 + 6}\quad \text{and}\quad X_2(s) = \frac{2/5}{s^2 + 1} + \frac{3/5}{s^2 + 6}. \end{equation*}
    Taking the inverse Laplace transform yields
    \begin{equation*} \begin{cases} x_1(t) &= \frac{1}{5} \sin t - \frac{6}{5 \sqrt{6}} \sin(\sqrt{6} t), \\ x_2(t) &= \frac{2}{5} \sin t + \frac{3}{5 \sqrt{6}} \sin(\sqrt{6} t), \end{cases} \end{equation*}
    which render the solutions of the given initial value problem (5.6.4).
    🔗
    🔗

    Subsection 5.6.5 Electrical Network Systems

    We discussed series and parallel \(LRC\) circuits in Chapter 3. In Figure 5.6.5 below, we have an \(LRC\) circuit with a voltage source \(e(t)\text{.}\) We develop a system of linear differential equations to find the charge \(q_1(t)\) on the capacitor and currents \(i(t)\) and \(i_2(t)\) flowing through the resistor and inductor, respectively. Let \(i_1(t)\) be the current flowing through the capacitor.
    🔗
    We observe that the circuit contains three distinct closed loops:
    🔗
    • Loop 1: consisting of the battery, resistor, and capacitor;
      🔗
      🔗
    • Loop 2: consisting of the battery, resistor, and inductor; and
      🔗
      🔗
    • Loop 3: formed by the capacitor and inductor alone.
      🔗
      🔗
    🔗
    The system of corresponding equations for the loops are:
    \begin{equation*} \begin{cases} \ds R\, i(t)+\frac{1}{C} q_1(t) = e(t) & \qquad\text{(for Loop 1)}\\ \ds L\, i_2'(t) + R\, i(t) = e(t) & \qquad \text{(for Loop 2)} \\ \ds L\, i_2'(t) - \frac{1}{C} q_1(t) = 0& \qquad \text{(for Loop 3)} \end{cases} \end{equation*}
    We observe that each equation in the system can be obtained from the remaining two. For example, the equation for Loop 2 can be obtained by adding the equations for Loop 1 and Loop 3. For the discussion that follows, we omit the equation for Loop 2.
    🔗
    Furthermore, by Kirchhoff’s second law, the current split at the junction A is given by
    \begin{equation*} i(t) = i_1(t) + i_2(t), \end{equation*}
    which, in view of \(i_1(t)= q_1'(t)\text{,}\) becomes
    \begin{equation*} q'_1(t) + i_2(t) -i(t) =0. \end{equation*}
    Thus, a linear system in \(i(t), q_1(t)\) and \(i_2(t)\) is
    \begin{equation} \begin{cases} \ds R\, i(t)+\frac{1}{C} q_1(t) = e(t) & \qquad\text{(for Loop 1)}\\ \ds L\, i_2'(t) - \frac{1}{C} q_1(t) = 0& \qquad \text{(for Loop 3)}\\ \ds q'_1(t) + i_2(t) -i(t) =0 & \qquad \text{(the current split at the junction A)} \end{cases}\tag{5.6.5} \end{equation}
    🔗
    We now proceed to solve the system (5.6.5). Differentiating the third equation in (5.6.5) with respect to \(t\) yields
    \begin{equation} i'(t) = q''_1(t) + i'_2(t).\tag{5.6.6} \end{equation}
    Solving for \(i'_2(t)\) from the second equation in (5.6.5) and substituting it into (5.6.6) gives
    \begin{equation*} i'(t) = q''_1(t) + \frac{1}{LC}q_1(t). \end{equation*}
    Substituting this expression for \(i'(t)\) into the equation obtained by differentiating the first equation in (5.6.5) with respect to \(t\text{,}\) we get
    \begin{equation} LRC q''_1(t) +Lq'_1(t)+ Rq_1(t) = LCe'(t),\tag{5.6.7} \end{equation}
    which can be solved for \(q_1(t).\) Once \(q_1(t)\) is determined, we find \(i_1(t)\) by using \(i_1(t)= q_1'(t).\) We then find \(i(t)\) from the first equation in (5.6.5) by using \(q_1(t)\) there. Finally, we find \(i_2(t) = i(t) - i_1(t).\)
